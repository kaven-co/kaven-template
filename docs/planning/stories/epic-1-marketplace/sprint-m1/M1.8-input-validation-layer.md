# M1.8 — Input Validation Layer

> **Epic:** 1 (Marketplace) | **Sprint:** M1 | **Agent:** @dev
> **Priority:** P1
> **Depends on:** M1.6 (módulos precisam de validação)

---

## Contexto

O Fastify com Zod type provider já está configurado, mas nem todas as rotas usam validação consistente. Precisamos garantir que TODA entrada (body, params, querystring, headers) é validada.

## User Story

**Como** API,
**Quero** rejeitar inputs inválidos antes de processar qualquer lógica,
**Para que** dados malformados nunca cheguem ao banco de dados.

---

## Acceptance Criteria

- [ ] Toda rota tem `schema.body`, `schema.params`, e/ou `schema.querystring` definidos com Zod
- [ ] Erros de validação retornam 422 com detalhes dos campos inválidos
- [ ] Params com UUID são validados com `z.string().uuid()`
- [ ] Slugs validados com regex: `z.string().regex(/^[a-z0-9-]+$/)`
- [ ] Pagination params com defaults: `page=1, limit=20, maxLimit=100`
- [ ] Body size limit configurado (1MB default, 10MB para uploads)
- [ ] Content-Type enforcement (application/json obrigatório para POST/PUT)
- [ ] Sanitização: strip HTML de strings, trim whitespace
- [ ] Schemas exportados como tipos TypeScript para reuso

---

## Technical Notes

### Schemas compartilhados:
```typescript
// src/schemas/common.ts
import { z } from 'zod';

export const paginationSchema = z.object({
  page: z.coerce.number().int().min(1).default(1),
  limit: z.coerce.number().int().min(1).max(100).default(20),
});

export const slugParamSchema = z.object({
  slug: z.string().regex(/^[a-z0-9]([a-z0-9-]*[a-z0-9])?$/, 'Invalid slug format'),
});

export const uuidParamSchema = z.object({
  id: z.string().uuid('Invalid UUID format'),
});

export const sortSchema = z.enum(['newest', 'oldest', 'popular', 'name']).default('newest');
```

### Schemas por recurso:
```typescript
// src/schemas/module.schemas.ts
export const createModuleSchema = z.object({
  slug: slugParamSchema.shape.slug,
  name: z.string().min(2).max(100).trim(),
  description: z.string().max(1000).trim().optional(),
  category: z.string().max(50).trim().optional(),
  isPublic: z.boolean().default(true),
});

export const updateModuleSchema = createModuleSchema.partial().omit({ slug: true });

// src/schemas/release.schemas.ts
export const createReleaseSchema = z.object({
  version: z.string().regex(/^\d+\.\d+\.\d+(-[a-zA-Z0-9.]+)?$/),
  artifactUrl: z.string().url(),
  signature: z.string().min(1),
  changelog: z.string().max(5000).optional(),
  minKavenVersion: z.string().optional(),
});
```

### Aplicar nas rotas (pattern):
```typescript
app.withTypeProvider<ZodTypeProvider>().get('/modules', {
  schema: {
    querystring: paginationSchema.merge(z.object({
      category: z.string().optional(),
      sort: sortSchema,
    })),
    response: { 200: moduleListResponseSchema }
  }
}, handler);
```
