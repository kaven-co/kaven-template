# M1.2 — GitHub OAuth Provider

> **Epic:** 1 (Marketplace) | **Sprint:** M1 | **Agent:** @dev
> **Priority:** P0 — necessário para autenticar usuários no browser
> **Depends on:** Nenhum (pode rodar em paralelo com M1.1)

---

## Contexto

O schema do marketplace já tem `User.githubId` (String @unique), mas não há nenhuma implementação de OAuth. Quando o usuário acessa a `verification_uri` para aprovar o device code (M1.1), ele precisa estar autenticado via GitHub OAuth.

## User Story

**Como** desenvolvedor,
**Quero** me autenticar no marketplace usando minha conta GitHub,
**Para que** minha identidade seja verificada e eu possa aprovar device codes e gerenciar licenças.

---

## Acceptance Criteria

- [ ] `GET /auth/github` redireciona para GitHub OAuth authorize URL
- [ ] `GET /auth/github/callback` recebe o code, troca por access token, busca user info
- [ ] Se o user já existe (por githubId), faz login e gera JWT
- [ ] Se o user não existe, cria automaticamente e gera JWT
- [ ] O JWT contém: `sub` (userId), `role` (USER/ADMIN), `email`, `githubId`
- [ ] Após login, redireciona para frontend com JWT em query param ou cookie seguro
- [ ] GitHub OAuth app ID e secret são configuráveis via env vars
- [ ] Erros de OAuth retornam mensagem amigável (não stack trace)

## Technical Notes

### Dependência a instalar:
```bash
pnpm add @fastify/oauth2
```

### Env vars necessárias:
```env
GITHUB_CLIENT_ID=xxx
GITHUB_CLIENT_SECRET=xxx
GITHUB_CALLBACK_URL=http://localhost:3000/auth/github/callback
```

### Novos arquivos:
```
src/routes/github-oauth.routes.ts   — Rotas OAuth
src/services/user.service.ts        — CRUD de users (findOrCreate)
```

### Flow:
```
1. Browser → GET /auth/github
2. Redirect → github.com/login/oauth/authorize?client_id=X&scope=user:email
3. User aprova → GitHub redirect → GET /auth/github/callback?code=XXX
4. Server troca code → access token via GitHub API
5. Server busca user info (GET /user) → email, id, avatar
6. Server findOrCreate user no DB
7. Server gera JWT
8. Redirect → frontend com token (cookie httpOnly ou ?token=xxx)
```

### Integração com @fastify/oauth2:
```typescript
import oauthPlugin from '@fastify/oauth2';

server.register(oauthPlugin, {
  name: 'githubOAuth2',
  credentials: {
    client: {
      id: process.env.GITHUB_CLIENT_ID!,
      secret: process.env.GITHUB_CLIENT_SECRET!,
    },
    auth: oauthPlugin.GITHUB_CONFIGURATION,
  },
  startRedirectPath: '/auth/github',
  callbackUri: process.env.GITHUB_CALLBACK_URL!,
  scope: ['user:email'],
});
```
