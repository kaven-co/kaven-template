# M2.4 — Paddle Webhook Handler

> **Epic:** 1 (Marketplace) | **Sprint:** M2 | **Agent:** @architect
> **Priority:** P0
> **Depends on:** M2.1

---

## Contexto

Paddle is the primary payment gateway for the Kaven Marketplace. When a user subscribes to a tier (Starter, Complete, Pro) through Paddle Checkout, Paddle sends webhook events to our server to notify us of subscription lifecycle changes. These webhooks are the source of truth for license creation, activation, cancellation, and revocation.

This is a critical integration point: if webhooks fail silently, users pay but never receive access. If they process duplicates, users get double licenses. The handler must be bulletproof: verify signatures, handle all event types, and guarantee idempotent processing.

The WebhookEvent model stores every received event for audit trail, debugging, and idempotency checks.

## User Story

**Como** sistema de marketplace, **Quero** processar webhooks do Paddle de forma confiável, **Para que** licenças sejam criadas, ativadas e revogadas automaticamente quando pagamentos acontecem.

---

## Acceptance Criteria

- [ ] `POST /webhooks/paddle` endpoint registered and accessible (no auth — Paddle calls it)
- [ ] Paddle webhook signature verified using HMAC SHA-256 with `PADDLE_WEBHOOK_SECRET`
- [ ] Invalid signature returns 403 and logs the attempt
- [ ] `subscription_created` event creates a new License with status `PENDING` and correct tier
- [ ] `transaction_completed` event activates the license (status → `ACTIVE`, sets `expiresAt`)
- [ ] `subscription_updated` event updates license tier if plan changed (upgrade/downgrade)
- [ ] `subscription_cancelled` event marks license status as `EXPIRED` at subscription period end
- [ ] `transaction_refunded` event revokes license immediately (status → `REVOKED`)
- [ ] All events are stored in WebhookEvent model with provider, eventId, eventType, payload
- [ ] Duplicate events (same eventId) are detected and skipped (return 200 without reprocessing)
- [ ] Paddle event_id is stored as unique field for idempotency
- [ ] Failed event processing does NOT return 500 (returns 200 to Paddle, logs error internally)
- [ ] Paddle tier mapping is configurable: `PADDLE_PRICE_STARTER` → `STARTER`, etc.
- [ ] Unknown event types are logged but acknowledged (return 200)
- [ ] Endpoint responds within 5 seconds (Paddle timeout)
- [ ] User lookup by Paddle customer email, with fallback to passthrough userId

### New Prisma Model

```prisma
model WebhookEvent {
  id          String    @id @default(cuid())
  provider    String    /// "paddle" | "stripe" — extensible for future gateways
  eventId     String    @unique /// Paddle event_id — idempotency key
  eventType   String    /// subscription_created, transaction_completed, etc.
  payload     Json      /// Full webhook payload for debugging
  processedAt DateTime? /// null if failed/skipped
  error       String?   /// Error message if processing failed
  createdAt   DateTime  @default(now())

  @@index([provider, eventType])
  @@index([createdAt])
}
```

### Paddle Webhook Signature Verification

```typescript
// src/utils/paddle-signature.ts

import crypto from 'node:crypto';

export function verifyPaddleWebhookSignature(
  rawBody: string | Buffer,
  signature: string,
  secret: string
): boolean {
  // Paddle Billing uses ts;h1=hash format
  const parts = signature.split(';');
  const tsParam = parts.find((p) => p.startsWith('ts='));
  const h1Param = parts.find((p) => p.startsWith('h1='));

  if (!tsParam || !h1Param) return false;

  const ts = tsParam.replace('ts=', '');
  const h1 = h1Param.replace('h1=', '');

  // Reconstruct signed payload: ts:rawBody
  const signedPayload = `${ts}:${rawBody}`;
  const expectedSignature = crypto
    .createHmac('sha256', secret)
    .update(signedPayload)
    .digest('hex');

  return crypto.timingSafeEqual(
    Buffer.from(h1, 'hex'),
    Buffer.from(expectedSignature, 'hex')
  );
}
```

### Webhook Handler Service

```typescript
// src/services/paddle-webhook.service.ts

export class PaddleWebhookService {
  constructor(
    private prisma: PrismaClient,
    private licenseService: LicenseService,
    private licenseKeyService: LicenseKeyService,
    private config: PaddleConfig
  ) {}

  async handleEvent(event: PaddleWebhookEvent): Promise<void> {
    // 1. Check idempotency
    const existing = await this.prisma.webhookEvent.findUnique({
      where: { eventId: event.event_id },
    });
    if (existing?.processedAt) {
      console.log(`Skipping duplicate event: ${event.event_id}`);
      return;
    }

    // 2. Store event
    const webhookEvent = await this.prisma.webhookEvent.upsert({
      where: { eventId: event.event_id },
      create: {
        provider: 'paddle',
        eventId: event.event_id,
        eventType: event.event_type,
        payload: event as any,
      },
      update: {},
    });

    // 3. Process by type
    try {
      switch (event.event_type) {
        case 'subscription.created':
          await this.handleSubscriptionCreated(event);
          break;
        case 'transaction.completed':
          await this.handleTransactionCompleted(event);
          break;
        case 'subscription.updated':
          await this.handleSubscriptionUpdated(event);
          break;
        case 'subscription.canceled':
          await this.handleSubscriptionCancelled(event);
          break;
        case 'transaction.refunded':
          await this.handleTransactionRefunded(event);
          break;
        default:
          console.log(`Unhandled Paddle event type: ${event.event_type}`);
      }

      // 4. Mark as processed
      await this.prisma.webhookEvent.update({
        where: { id: webhookEvent.id },
        data: { processedAt: new Date() },
      });
    } catch (error) {
      // Log error but don't throw — always return 200 to Paddle
      console.error(`Failed to process event ${event.event_id}:`, error);
      await this.prisma.webhookEvent.update({
        where: { id: webhookEvent.id },
        data: { error: (error as Error).message },
      });
    }
  }

  private async handleSubscriptionCreated(event: PaddleWebhookEvent): Promise<void> {
    const { data } = event;
    const priceId = data.items?.[0]?.price?.id;
    const tier = this.config.priceToTier[priceId];
    if (!tier) throw new Error(`Unknown Paddle price ID: ${priceId}`);

    // Find user by email or passthrough
    const user = await this.resolveUser(data.customer?.email, data.custom_data?.userId);
    if (!user) throw new Error(`User not found for event ${event.event_id}`);

    // Generate license key
    const key = this.licenseKeyService.generate(tier);

    await this.licenseService.create({
      key,
      type: tier,
      status: 'PENDING', // Activated on transaction_completed
      userId: user.id,
      paddleSubscriptionId: data.id,
    });
  }

  private async handleTransactionCompleted(event: PaddleWebhookEvent): Promise<void> {
    const subscriptionId = event.data.subscription_id;
    if (!subscriptionId) return; // One-time purchase, handle separately

    const license = await this.prisma.license.findFirst({
      where: { paddleSubscriptionId: subscriptionId },
      orderBy: { createdAt: 'desc' },
    });
    if (!license) throw new Error(`License not found for subscription ${subscriptionId}`);

    // Calculate expiry from billing period
    const periodEnd = event.data.billing_period?.ends_at;
    const expiresAt = periodEnd ? new Date(periodEnd) : null;

    await this.prisma.license.update({
      where: { id: license.id },
      data: {
        status: 'ACTIVE',
        expiresAt,
        activatedAt: new Date(),
      },
    });
  }

  private async handleSubscriptionUpdated(event: PaddleWebhookEvent): Promise<void> {
    const subscriptionId = event.data.id;
    const newPriceId = event.data.items?.[0]?.price?.id;
    const newTier = this.config.priceToTier[newPriceId];

    if (!newTier) return; // Not a tier change we recognize

    const license = await this.prisma.license.findFirst({
      where: { paddleSubscriptionId: subscriptionId, status: 'ACTIVE' },
    });
    if (!license) return;

    // Regenerate key with new tier prefix
    const newKey = this.licenseKeyService.generate(newTier);

    await this.prisma.license.update({
      where: { id: license.id },
      data: {
        type: newTier,
        key: newKey,
      },
    });
  }

  private async handleSubscriptionCancelled(event: PaddleWebhookEvent): Promise<void> {
    const subscriptionId = event.data.id;
    const effectiveAt = event.data.scheduled_change?.effective_at;

    const license = await this.prisma.license.findFirst({
      where: { paddleSubscriptionId: subscriptionId, status: 'ACTIVE' },
    });
    if (!license) return;

    await this.prisma.license.update({
      where: { id: license.id },
      data: {
        status: 'EXPIRED',
        expiresAt: effectiveAt ? new Date(effectiveAt) : new Date(),
      },
    });
  }

  private async handleTransactionRefunded(event: PaddleWebhookEvent): Promise<void> {
    const subscriptionId = event.data.subscription_id;
    if (!subscriptionId) return;

    const license = await this.prisma.license.findFirst({
      where: { paddleSubscriptionId: subscriptionId },
    });
    if (!license) return;

    await this.licenseService.revoke(license.id, 'SYSTEM_REFUND');
  }

  private async resolveUser(email?: string, userId?: string): Promise<User | null> {
    if (userId) {
      return this.prisma.user.findUnique({ where: { id: userId } });
    }
    if (email) {
      return this.prisma.user.findUnique({ where: { email } });
    }
    return null;
  }
}
```

### Route Registration

```typescript
// src/routes/webhooks.route.ts

const webhooksRoute: FastifyPluginAsync = async (fastify) => {
  // Need raw body for signature verification
  fastify.addContentTypeParser(
    'application/json',
    { parseAs: 'string' },
    (req, body, done) => {
      try {
        const json = JSON.parse(body as string);
        (req as any).rawBody = body;
        done(null, json);
      } catch (err) {
        done(err as Error);
      }
    }
  );

  fastify.post('/paddle', {
    config: { rateLimit: false }, // No rate limit for webhooks
    handler: async (request, reply) => {
      const signature = request.headers['paddle-signature'] as string;
      if (!signature) return reply.code(403).send({ error: 'Missing signature' });

      const valid = verifyPaddleWebhookSignature(
        (request as any).rawBody,
        signature,
        paddleConfig.webhookSecret
      );
      if (!valid) {
        console.warn('Invalid Paddle webhook signature');
        return reply.code(403).send({ error: 'Invalid signature' });
      }

      // Process asynchronously but return 200 immediately
      await paddleWebhookService.handleEvent(request.body as PaddleWebhookEvent);
      return reply.code(200).send({ received: true });
    },
  });
};
```
