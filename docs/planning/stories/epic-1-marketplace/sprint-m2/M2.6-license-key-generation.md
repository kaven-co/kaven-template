# M2.6 — License Key Generation

> **Epic:** 1 (Marketplace) | **Sprint:** M2 | **Agent:** @dev
> **Priority:** P0
> **Depends on:** M2.1

---

## Contexto

License keys are the user-facing representation of a license. They serve two purposes: human-readable identification (a user can look at `KAVEN-PRO-WDKF7H3N-X2` and know they have a Pro license) and offline validation (the CLI can validate the key format and checksum without hitting the API).

The key format is designed to be:
1. **Recognizable** — starts with `KAVEN-` prefix
2. **Informative** — tier code is embedded (`STR`, `CMP`, `PRO`, `ENT`)
3. **Unambiguous** — excludes confusing characters (0/O, 1/I/L)
4. **Verifiable** — Luhn mod-36 checksum enables offline format validation
5. **Unique** — enforced at database level

Keys are generated automatically by M2.4 (Paddle webhook on subscription_created) or manually by admins for enterprise/trial licenses.

## User Story

**Como** developer com licença Kaven, **Quero** ter uma chave de licença legível e verificável, **Para que** eu possa identificar meu plano e o CLI possa validar o formato sem precisar de internet.

---

## Acceptance Criteria

- [ ] Key format: `KAVEN-{TIER}-{8RANDOM}-{CHECKSUM}`
- [ ] Tier codes: `STR` (Starter), `CMP` (Complete), `PRO` (Pro), `ENT` (Enterprise)
- [ ] Random segment: 8 uppercase alphanumeric characters from safe alphabet
- [ ] Safe alphabet excludes: `0`, `O`, `1`, `I`, `L` (26 letters + 10 digits - 5 = 31 chars)
- [ ] Checksum: 2-character Luhn mod-36 of the key body (everything after `KAVEN-` and before checksum)
- [ ] `LicenseKeyService.generate(tier)` returns a valid, formatted key
- [ ] `LicenseKeyService.validate(key)` returns `{ valid: boolean, reason?: string }`
- [ ] `LicenseKeyService.parse(key)` returns `{ tier: LicenseType, body: string, checksum: string }` or throws
- [ ] validate() checks: correct prefix, valid tier code, correct character set, valid checksum
- [ ] Generated keys are unique (collision probability < 1 in 10^12 given 31^8 = 852 billion combos)
- [ ] Key generation retries up to 3 times on DB uniqueness conflict
- [ ] Example valid key: `KAVEN-PRO-WDKF7H3N-X2`
- [ ] Example invalid keys rejected: `KAVEN-XXX-AAAAAAAA-00`, `KAVEN-PRO-00000000-00`, `kaven-pro-wdkf7h3n-x2`

## Technical Notes

### Safe Alphabet

```typescript
// src/utils/license-key.ts

// 31 characters: A-Z minus O,I,L + 0-9 minus 0,1
const SAFE_ALPHABET = 'ABCDEFGHJKMNPQRSTUVWXYZ23456789';
// Index: A=0, B=1, ..., 9=30
```

### Luhn Mod-36 Checksum

```typescript
// src/utils/license-key.ts

const CHECKSUM_ALPHABET = SAFE_ALPHABET; // Same 31-char set but using mod-31

/**
 * Compute Luhn mod-N checksum (adapted for our 31-char alphabet).
 * Returns a 2-character checksum string.
 *
 * Algorithm (Luhn mod-N with N=31):
 * 1. Process characters right-to-left
 * 2. Even positions: double the code point, if >= N, subtract N-1
 * 3. Sum all code points
 * 4. Checksum char = (N - (sum % N)) % N
 * 5. Second char: repeat on body + first checksum char
 */
function computeChecksum(body: string): string {
  const N = SAFE_ALPHABET.length; // 31

  function luhnChar(input: string): string {
    let sum = 0;
    let double = true;

    for (let i = input.length - 1; i >= 0; i--) {
      let codePoint = SAFE_ALPHABET.indexOf(input[i]);
      if (codePoint === -1) continue; // Skip separators

      if (double) {
        codePoint *= 2;
        if (codePoint >= N) codePoint -= (N - 1);
      }
      sum += codePoint;
      double = !double;
    }

    const remainder = sum % N;
    return SAFE_ALPHABET[(N - remainder) % N];
  }

  // Strip non-alphabet chars (dashes) for computation
  const stripped = body.replace(/-/g, '').replace(/^KAVEN/, '');
  const check1 = luhnChar(stripped);
  const check2 = luhnChar(stripped + check1);

  return check1 + check2;
}
```

### LicenseKeyService

```typescript
// src/services/license-key.service.ts

import crypto from 'node:crypto';

const TIER_CODES: Record<LicenseType, string> = {
  STARTER: 'STR',
  COMPLETE: 'CMP',
  PRO: 'PRO',
  ENTERPRISE: 'ENT',
};

const CODE_TO_TIER: Record<string, LicenseType> = Object.fromEntries(
  Object.entries(TIER_CODES).map(([k, v]) => [v, k as LicenseType])
);

const KEY_REGEX = /^KAVEN-(STR|CMP|PRO|ENT)-([ABCDEFGHJKMNPQRSTUVWXYZ23456789]{8})-([ABCDEFGHJKMNPQRSTUVWXYZ23456789]{2})$/;

export class LicenseKeyService {
  constructor(private prisma: PrismaClient) {}

  generate(tier: LicenseType): string {
    const tierCode = TIER_CODES[tier];
    if (!tierCode) throw new Error(`Invalid tier: ${tier}`);

    // Generate 8 random characters from safe alphabet
    const random = this.generateRandomSegment(8);

    // Compute body for checksum: tier code + random
    const body = `${tierCode}${random}`;
    const checksum = computeChecksum(body);

    return `KAVEN-${tierCode}-${random}-${checksum}`;
  }

  validate(key: string): KeyValidationResult {
    // 1. Format check
    const match = key.match(KEY_REGEX);
    if (!match) {
      return { valid: false, reason: 'INVALID_FORMAT' };
    }

    const [, tierCode, random, providedChecksum] = match;

    // 2. Tier check
    if (!CODE_TO_TIER[tierCode]) {
      return { valid: false, reason: 'INVALID_TIER' };
    }

    // 3. Character set check (already covered by regex, but explicit)
    for (const char of random) {
      if (!SAFE_ALPHABET.includes(char)) {
        return { valid: false, reason: 'INVALID_CHARACTERS' };
      }
    }

    // 4. Checksum verification
    const body = `${tierCode}${random}`;
    const expectedChecksum = computeChecksum(body);
    if (providedChecksum !== expectedChecksum) {
      return { valid: false, reason: 'INVALID_CHECKSUM' };
    }

    return { valid: true };
  }

  parse(key: string): ParsedKey {
    const validation = this.validate(key);
    if (!validation.valid) {
      throw new Error(`Invalid license key: ${validation.reason}`);
    }

    const match = key.match(KEY_REGEX)!;
    const [, tierCode, body, checksum] = match;

    return {
      tier: CODE_TO_TIER[tierCode],
      tierCode,
      body,
      checksum,
      full: key,
    };
  }

  /**
   * Generate key with DB uniqueness guarantee.
   * Retries up to 3 times on collision.
   */
  async generateUnique(tier: LicenseType, maxRetries = 3): Promise<string> {
    for (let attempt = 0; attempt < maxRetries; attempt++) {
      const key = this.generate(tier);
      const existing = await this.prisma.license.findUnique({
        where: { key },
        select: { id: true },
      });
      if (!existing) return key;
      console.warn(`License key collision on attempt ${attempt + 1}, retrying...`);
    }
    throw new Error('Failed to generate unique license key after max retries');
  }

  private generateRandomSegment(length: number): string {
    const bytes = crypto.randomBytes(length);
    let result = '';
    for (let i = 0; i < length; i++) {
      result += SAFE_ALPHABET[bytes[i] % SAFE_ALPHABET.length];
    }
    return result;
  }
}
```

### Usage in CLI (Offline Validation)

```typescript
// kaven-cli can validate format without API call:
import { LicenseKeyService } from './license-key.service';

const keyService = new LicenseKeyService(null as any); // No prisma needed for validate/parse
const result = keyService.validate('KAVEN-PRO-WDKF7H3N-X2');

if (result.valid) {
  const parsed = keyService.parse('KAVEN-PRO-WDKF7H3N-X2');
  console.log(`Valid ${parsed.tier} license key`);
  // Now call API for full validation (status, expiry, etc.)
} else {
  console.error(`Invalid key format: ${result.reason}`);
}
```
