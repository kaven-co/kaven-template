# M2.5 — Paddle Checkout Integration

> **Epic:** 1 (Marketplace) | **Sprint:** M2 | **Agent:** @architect
> **Priority:** P1
> **Depends on:** M2.4

---

## Contexto

While M2.4 handles the backend webhook processing (what happens after payment), this story handles the frontend-facing payment initiation (what happens before payment). Users need a way to select a tier and be redirected to Paddle's hosted checkout page to complete payment.

Paddle Checkout is a hosted payment page — we create a "checkout session" via Paddle's API with the customer's details and desired plan, then redirect them to Paddle's URL. After payment, Paddle redirects back to our success page and fires webhooks (handled by M2.4).

This story also provides a customer portal URL so existing subscribers can manage their subscription (upgrade, downgrade, cancel) through Paddle's self-service portal.

## User Story

**Como** developer que quer comprar uma licença Kaven, **Quero** ser redirecionado para um checkout seguro onde posso pagar com cartão ou PayPal, **Para que** eu receba minha licença automaticamente após o pagamento.

- [ ] `POST /checkout/session` accepts `{ tier: 'STARTER' | 'COMPLETE' | 'PRO' }`

- [ ] Includes `custom_data` passthrough: `{ userId, tier }`
- [ ] Returns `{ checkoutUrl: string, transactionId: string }`
- [ ] `GET /checkout/success` serves a success callback page (redirected from Paddle)
- [ ] Success page displays "Payment processing..." with polling to verify license activation
- [ ] `POST /checkout/verify` accepts `{ transactionId: string }` and returns payment status
- [ ] Verify endpoint checks Paddle API for transaction status and returns `{ status, license? }`
- [ ] `GET /checkout/portal` returns Paddle customer portal URL for managing subscription
- [ ] If user already has an active license of the same or higher tier, return 409 Conflict
- [ ] If user has a lower-tier license, allow checkout (upgrade path)
- [ ] Invalid tier returns 400 Bad Request
- [ ] All Paddle API calls use `PADDLE_API_KEY` and respect `PADDLE_ENVIRONMENT` (sandbox/production)

## Technical Notes

### PaddleService Implementation

```typescript
// src/services/paddle.service.ts

import { Paddle, Environment } from '@paddle/paddle-node-sdk';

export class PaddleService {
  private paddle: Paddle;
  private config: PaddleConfig;

  constructor(config: PaddleConfig) {
    this.config = config;
    this.paddle = new Paddle(config.apiKey, {
      environment: config.environment === 'sandbox'
        ? Environment.sandbox
        : Environment.production,
    });
  }

  async createCheckoutSession(
    userId: string,
    email: string,
    tier: 'STARTER' | 'COMPLETE' | 'PRO'
  ): Promise<CheckoutSession> {
    const priceId = this.getTierPriceId(tier);

    const transaction = await this.paddle.transactions.create({
      items: [
        {
          priceId,
          quantity: 1,
        },
      ],
      customData: {
        userId,
        tier,
      },
      customer: {
        email,
      },
      checkout: {
        url: `${process.env.APP_URL}/checkout/success?transaction_id={transaction_id}`,
      },
    });

    return {
      checkoutUrl: transaction.checkout?.url || '',
      transactionId: transaction.id,
    };
  }

  async verifyTransaction(transactionId: string): Promise<TransactionStatus> {
    const transaction = await this.paddle.transactions.get(transactionId);

    return {
      status: transaction.status, // 'draft' | 'ready' | 'billed' | 'paid' | 'completed' | 'canceled'
      customData: transaction.customData as { userId: string; tier: string } | undefined,
      subscriptionId: transaction.subscriptionId || undefined,
    };
  }

  async getCustomerPortalUrl(customerId: string): Promise<string> {
    // Paddle customer portal is accessed via a session URL
    const session = await this.paddle.customers.generateAuthToken(customerId);
    return `https://${this.config.environment === 'sandbox' ? 'sandbox-' : ''}customer-portal.paddle.com/?token=${session}`;
  }

  private getTierPriceId(tier: 'STARTER' | 'COMPLETE' | 'PRO'): string {
    const mapping: Record<string, string> = {
      STARTER: process.env.PADDLE_PRICE_STARTER!,
      COMPLETE: process.env.PADDLE_PRICE_COMPLETE!,
      PRO: process.env.PADDLE_PRICE_PRO!,
    };

    const priceId = mapping[tier];
    if (!priceId) throw new BadRequestError(`Invalid tier: ${tier}`);
    return priceId;
  }
}
```

### Route Definitions

```typescript
// src/routes/checkout.route.ts

const checkoutRoute: FastifyPluginAsync = async (fastify) => {
  // POST /checkout/session — create checkout
  fastify.post('/session', {
    preHandler: [fastify.authenticate],
    schema: {
      body: {
        type: 'object',
        required: ['tier'],
        properties: {
          tier: { type: 'string', enum: ['STARTER', 'COMPLETE', 'PRO'] },
        },
      },
      response: {
        201: {
          type: 'object',
          properties: {
            checkoutUrl: { type: 'string' },
            transactionId: { type: 'string' },
          },
        },
      },
    },
    handler: async (request, reply) => {
      const { tier } = request.body as { tier: 'STARTER' | 'COMPLETE' | 'PRO' };
      const user = request.user;

      // Check for existing license conflict
      const existingLicenses = await licenseService.getByUser(user.id, { status: 'ACTIVE' });
      const hasEqualOrHigherTier = existingLicenses.some(
        (l) => TIER_HIERARCHY[l.type] >= TIER_HIERARCHY[tier]
      );
      if (hasEqualOrHigherTier) {
        return reply.code(409).send({
          error: 'EXISTING_LICENSE',
          message: 'You already have an active license of this tier or higher',
        });
      }

      const session = await paddleService.createCheckoutSession(
        user.id,
        user.email,
        tier
      );
      return reply.code(201).send(session);
    },
  });

  // GET /checkout/success — callback from Paddle
  fastify.get('/success', {
    schema: {
      querystring: {
        type: 'object',
        properties: {
          transaction_id: { type: 'string' },
        },
      },
    },
    handler: async (request, reply) => {
      const { transaction_id } = request.query as { transaction_id?: string };

      // Return a simple HTML page that polls /checkout/verify
      reply.type('text/html').send(`
        <!DOCTYPE html>
        <html>
        <head><title>Payment Processing</title></head>
        <body>
          <h1>Payment Processing...</h1>
          <p id="status">Verifying your payment. This may take a few seconds.</p>
          <script>
            const txnId = '${transaction_id || ''}';
            async function checkStatus() {
              if (!txnId) { document.getElementById('status').textContent = 'No transaction ID'; return; }
              try {
                const res = await fetch('/checkout/verify', {
                  method: 'POST',
                  headers: { 'Content-Type': 'application/json' },
                  body: JSON.stringify({ transactionId: txnId }),
                });
                const data = await res.json();
                if (data.status === 'completed' || data.status === 'paid') {
                  document.getElementById('status').textContent =
                    'Payment confirmed! Your license key: ' + (data.license?.key || 'Processing...');
                } else {
                  setTimeout(checkStatus, 3000);
                }
              } catch (e) { setTimeout(checkStatus, 3000); }
            }
            checkStatus();
          </script>
        </body>
        </html>
      `);
    },
  });

  // POST /checkout/verify — check transaction status
  fastify.post('/verify', {
    schema: {
      body: {
        type: 'object',
        required: ['transactionId'],
        properties: {
          transactionId: { type: 'string' },
        },
      },
    },
    handler: async (request, reply) => {
      const { transactionId } = request.body as { transactionId: string };
      const txnStatus = await paddleService.verifyTransaction(transactionId);

      let license = null;
      if (txnStatus.customData?.userId && txnStatus.subscriptionId) {
        license = await prisma.license.findFirst({
          where: {
            paddleSubscriptionId: txnStatus.subscriptionId,
            userId: txnStatus.customData.userId,
          },
          select: { key: true, type: true, status: true, expiresAt: true },
        });
      }

      return reply.send({
        status: txnStatus.status,
        license,
      });
    },
  });

  // GET /checkout/portal — customer self-service portal
  fastify.get('/portal', {
    preHandler: [fastify.authenticate],
    handler: async (request, reply) => {
      const user = request.user;

      if (!user.paddleCustomerId) {
        return reply.code(404).send({ error: 'No Paddle customer found' });
      }

      const portalUrl = await paddleService.getCustomerPortalUrl(user.paddleCustomerId);
      return reply.send({ portalUrl });
    },
  });
};
```
