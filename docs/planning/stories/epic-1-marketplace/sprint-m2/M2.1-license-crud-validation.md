# M2.1 — License CRUD & Validation

> **Epic:** 1 (Marketplace) | **Sprint:** M2 | **Agent:** @dev
> **Priority:** P0
> **Depends on:** M1.3

---

## Contexto

Licensing is the core monetization primitive of the Kaven Marketplace. Every paid module download, every tier upgrade, and every CLI activation depends on a license being created, validated, and enforced. Without this story, there is no paywall — modules would be freely downloadable regardless of subscription tier.

The License model already exists in the Prisma schema with `key`, `type` (LicenseType enum), `status` (LicenseStatus enum), `expiresAt`, and `userId`. This story builds the service layer and API routes on top of that schema, providing CRUD operations and a validation engine that checks status, expiry, and tier compatibility.

The hybrid pricing model means some modules are tier-gated (official Kaven modules require STARTER/COMPLETE/PRO), while community modules can be individually priced or free. The `requiredTier` field on Module determines which license tier is needed to access it.

## User Story

**Como** developer com licença Kaven, **Quero** que o sistema valide minha licença ao baixar módulos, **Para que** eu tenha acesso apenas aos módulos que meu plano permite.

---

## Acceptance Criteria

- [ ] `GET /licenses` returns all licenses for the authenticated user (paginated, sorted by createdAt desc)
- [ ] `GET /licenses` requires authentication (401 without valid JWT)
- [ ] `POST /licenses/validate` accepts `{ key: string, moduleSlug?: string }` and returns validation result
- [ ] Validation checks: license status is `ACTIVE`, `expiresAt` is in the future (or null for lifetime)
- [ ] Validation checks: if `moduleSlug` is provided, license tier >= module's `requiredTier`
- [ ] Validation returns structured response: `{ valid: boolean, license?: LicensePublic, reason?: string }`
- [ ] `POST /licenses/:id/revoke` is admin-only (requires SUPER_ADMIN or MARKETPLACE_ADMIN role)
- [ ] Revoke sets license status to `REVOKED` and records `revokedAt` timestamp
- [ ] Revoking an already-revoked license returns 409 Conflict
- [ ] Tier hierarchy is enforced: PRO > COMPLETE > STARTER (PRO can access STARTER modules)
- [ ] `requiredTier: null` modules are accessible without any license (free modules)
- [ ] License list endpoint supports filtering by `status` query param
- [ ] All endpoints return proper error codes: 400 (bad input), 401 (no auth), 403 (wrong role), 404 (not found), 409 (conflict)
- [ ] License validation is idempotent (multiple calls with same key return same result)

## Technical Notes

### Tier Hierarchy

```typescript
// src/services/license.service.ts

const TIER_HIERARCHY: Record<LicenseType, number> = {
  STARTER: 1,
  COMPLETE: 2,
  PRO: 3,
  ENTERPRISE: 4,
};

function tierSatisfiesRequirement(
  licenseTier: LicenseType,
  requiredTier: LicenseType | null
): boolean {
  if (requiredTier === null) return true; // free module
  return TIER_HIERARCHY[licenseTier] >= TIER_HIERARCHY[requiredTier];
}
```

### LicenseService Interface

```typescript
// src/services/license.service.ts

export class LicenseService {
  constructor(
    private prisma: PrismaClient,
    private cache?: CacheService
  ) {}

  async create(data: CreateLicenseInput): Promise<License> { ... }

  async validate(key: string, moduleSlug?: string): Promise<ValidationResult> {
    const license = await this.prisma.license.findUnique({
      where: { key },
      include: { user: true },
    });

    if (!license) return { valid: false, reason: 'LICENSE_NOT_FOUND' };
    if (license.status !== 'ACTIVE') return { valid: false, reason: 'LICENSE_INACTIVE' };
    if (license.expiresAt && license.expiresAt < new Date()) {
      return { valid: false, reason: 'LICENSE_EXPIRED' };
    }

    if (moduleSlug) {
      const module = await this.prisma.module.findUnique({
        where: { slug: moduleSlug },
      });
      if (!module) return { valid: false, reason: 'MODULE_NOT_FOUND' };
      if (!tierSatisfiesRequirement(license.type, module.requiredTier)) {
        return { valid: false, reason: 'TIER_INSUFFICIENT' };
      }
    }

    return {
      valid: true,
      license: sanitizeLicense(license),
    };
  }

  async revoke(id: string, adminId: string): Promise<License> {
    const license = await this.prisma.license.findUnique({ where: { id } });
    if (!license) throw new NotFoundError('License not found');
    if (license.status === 'REVOKED') throw new ConflictError('License already revoked');

    return this.prisma.license.update({
      where: { id },
      data: {
        status: 'REVOKED',
        revokedAt: new Date(),
        revokedBy: adminId,
      },
    });
  }

  async getByUser(userId: string, filters?: LicenseFilters): Promise<License[]> { ... }
  async getByKey(key: string): Promise<License | null> { ... }
}
```

### Route Registration

```typescript
// src/routes/licenses.route.ts

import { FastifyPluginAsync } from 'fastify';

const licensesRoute: FastifyPluginAsync = async (fastify) => {
  // GET /licenses — user's licenses
  fastify.get('/', {
    preHandler: [fastify.authenticate],
    schema: {
      querystring: {
        type: 'object',
        properties: {
          status: { type: 'string', enum: ['ACTIVE', 'EXPIRED', 'REVOKED', 'SUSPENDED'] },
          page: { type: 'integer', minimum: 1, default: 1 },
          limit: { type: 'integer', minimum: 1, maximum: 100, default: 20 },
        },
      },
    },
    handler: async (request, reply) => {
      const { status, page, limit } = request.query as any;
      const licenses = await licenseService.getByUser(request.user.id, { status, page, limit });
      return reply.send(licenses);
    },
  });

  // POST /licenses/validate
  fastify.post('/validate', {
    schema: {
      body: {
        type: 'object',
        required: ['key'],
        properties: {
          key: { type: 'string', pattern: '^KAVEN-[A-Z]{3}-[A-Z0-9]{8}-[A-Z0-9]{2}$' },
          moduleSlug: { type: 'string' },
        },
      },
    },
    handler: async (request, reply) => {
      const { key, moduleSlug } = request.body as any;
      const result = await licenseService.validate(key, moduleSlug);
      return reply.send(result);
    },
  });

  // POST /licenses/:id/revoke — admin only
  fastify.post('/:id/revoke', {
    preHandler: [fastify.authenticate, fastify.requireRole('MARKETPLACE_ADMIN')],
    handler: async (request, reply) => {
      const { id } = request.params as any;
      const license = await licenseService.revoke(id, request.user.id);
      return reply.send(license);
    },
  });
};
```

### Validation Response Types

```typescript
// src/types/license.types.ts

export interface ValidationResult {
  valid: boolean;
  license?: LicensePublic;
  reason?: 'LICENSE_NOT_FOUND' | 'LICENSE_INACTIVE' | 'LICENSE_EXPIRED'
         | 'TIER_INSUFFICIENT' | 'MODULE_NOT_FOUND';
}

export interface LicensePublic {
  id: string;
  key: string;
  type: LicenseType;
  status: LicenseStatus;
  expiresAt: Date | null;
  createdAt: Date;
}

export interface LicenseFilters {
  status?: LicenseStatus;
  page?: number;
  limit?: number;
}
```
