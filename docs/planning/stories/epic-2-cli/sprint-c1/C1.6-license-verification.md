# C1.6 — License Verification

> **Epic:** 2 (CLI) | **Sprint:** C1 | **Agent:** @dev
> **Priority:** P1
> **Depends on:** C1.5 (Marketplace Install Real), M2.1 (Marketplace — License Validation endpoint)

---

## Contexto

When a user tries to install a tier-gated module, the CLI must verify that their license allows access. This verification should happen before the download begins (fail fast). The marketplace API provides a `POST /licenses/validate` endpoint that checks the user's active license against the module's tier requirements. Additionally, the CLI should cache successful license validations to reduce API calls and provide a `kaven license status` command for visibility.

For offline resilience, a basic local format check (Luhn checksum on license keys) can catch obviously invalid keys before making the API call.

## User Story

**As a** developer using kaven-cli,
**I want** the CLI to verify my license before installing tier-gated modules,
**So that** I get clear feedback about what I can access and how to upgrade if needed.

---

## Acceptance Criteria

- [ ] Before module install, call `POST /licenses/validate` with `{ moduleId, moduleSlug }`
- [ ] If license is valid: proceed with install silently
- [ ] If no valid license: display tier comparison table showing what each tier includes
- [ ] Tier comparison shows: user's current tier highlighted, required tier for the module
- [ ] Upgrade prompt after tier table: `"Upgrade to [tier]? Run: kaven upgrade [tier]"`
- [ ] If license is expired: show `"Your license expired on [date]. Renew at https://kaven.site/account"`
- [ ] Offline format validation: check license key format locally (Luhn checksum) before API call
- [ ] If local format check fails: show `"Invalid license key format"` without API call
- [ ] Cache successful validation in `~/.kaven/cache/licenses.json` with 1-hour TTL
- [ ] Cached validations include: `{ moduleSlug, tier, validatedAt, expiresAt }`
- [ ] Cache lookup before API call: if cached and not expired, skip API call
- [ ] `kaven license status` command shows:
  - Current tier name and description
  - License expiry date
  - List of accessible module categories
  - Number of modules available at current tier
- [ ] `kaven license status` when not authenticated: `"Not logged in. Run: kaven auth login"`

```typescript
    head: ['', ...Object.keys(TIERS).map(t =>

    ['Tenants', ...Object.values(TIERS).map(t => t.tenants)],
    ['Projects', ...Object.values(TIERS).map(t => t.projects)],
  );

  console.log('\n' + table.toString());
  console.log(
    chalk.yellow(`\n  Upgrade to ${TIERS[requiredTier].name}? Run: kaven upgrade ${requiredTier}\n`)
  );
}
```

### License Validation Service

```typescript
// src/core/LicenseService.ts
import * as fs from 'fs';
import * as path from 'path';
import * as os from 'os';

const CACHE_TTL_MS = 60 * 60 * 1000; // 1 hour

export class LicenseService {
  private cachePath: string;

  constructor() {
    this.cachePath = path.join(os.homedir(), '.kaven', 'cache', 'licenses.json');
  }

  /**
   * Validate license for a module. Uses cache first, then API.
   */
  async validate(
    client: MarketplaceClient,
    moduleSlug: string,
    moduleId: string
  ): Promise<LicenseValidationResult> {
    // 1. Check cache
    const cached = this.getCached(moduleSlug);
    if (cached) return { valid: true, tier: cached.tier, source: 'cache' };

    // 2. API validation
    try {
      const result = await client.validateLicense(moduleId, moduleSlug);
      if (result.valid) {
        this.cacheValidation(moduleSlug, result.tier);
      }
      return result;
    } catch (error) {
      if (error instanceof LicenseRequiredError) {
        return {
          valid: false,
          currentTier: error.currentTier,
          requiredTier: error.requiredTier,
          source: 'api',
        };
      }
      throw error;
    }
  }

  private getCached(slug: string): CachedLicense | null {
    try {
      const data = JSON.parse(fs.readFileSync(this.cachePath, 'utf-8'));
      const entry = data[slug];
      if (!entry) return null;
      if (new Date(entry.expiresAt).getTime() < Date.now()) return null;
      return entry;
    } catch {
      return null;
    }
  }

  private cacheValidation(slug: string, tier: string): void {
    const dir = path.dirname(this.cachePath);
    fs.mkdirSync(dir, { recursive: true });

    let data: Record<string, CachedLicense> = {};
    try {
      data = JSON.parse(fs.readFileSync(this.cachePath, 'utf-8'));
    } catch { /* empty cache */ }

    data[slug] = {
      moduleSlug: slug,
      tier,
      validatedAt: new Date().toISOString(),
      expiresAt: new Date(Date.now() + CACHE_TTL_MS).toISOString(),
      valid: true,
    };

    fs.writeFileSync(this.cachePath, JSON.stringify(data, null, 2));
  }

  /**
   * Get current license status from API.
   */
  async getStatus(client: MarketplaceClient): Promise<LicenseStatus> {
    return client.getLicenseStatus();
  }
}
```

### Luhn Checksum (local format validation)

```typescript
function isValidLicenseFormat(key: string): boolean {
  // Kaven license keys: KAVEN-XXXX-XXXX-XXXX-XXXX
  const pattern = /^KAVEN-[A-Z0-9]{4}-[A-Z0-9]{4}-[A-Z0-9]{4}-[A-Z0-9]{4}$/;
  if (!pattern.test(key)) return false;

  // Luhn checksum on numeric portion
  const digits = key.replace(/[^0-9]/g, '');
  if (digits.length < 4) return true; // Skip Luhn if mostly alpha
  return luhnCheck(digits);
}
```
