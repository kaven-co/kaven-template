# C1.3 — Token Management

> **Epic:** 2 (CLI) | **Sprint:** C1 | **Agent:** @dev
> **Priority:** P1
> **Depends on:** M1.3 (Marketplace Auth — Token Refresh endpoint)

---

## Contexto

The current `AuthService` stores a single mock JWT string and has no concept of token expiry, refresh tokens, or real user identity. For production use, the CLI needs proper JWT lifecycle management: decode tokens to read expiry and user info, auto-refresh before expiration, handle refresh failures gracefully, and provide `whoami` with real user data.

The CLI should never validate JWT signatures client-side (that is the server's job). It only decodes the payload to read `exp`, `email`, `githubId`, and `tier` claims.

## User Story

**As a** developer using kaven-cli,
**I want** my authentication to persist across sessions and auto-refresh transparently,
**So that** I don't have to re-login every time I use the CLI.

---

## Acceptance Criteria

- [ ] JWT payload decoded using base64 (no signature verification client-side)
- [ ] Token expiry read from `exp` claim (Unix timestamp)
- [ ] Auto-refresh triggered when token expires in less than 5 minutes
- [ ] Refresh calls `POST /auth/refresh` with `refresh_token` via `MarketplaceClient`
- [ ] On refresh success: update both `access_token` and `refresh_token` in `~/.kaven/auth.json`
- [ ] On refresh failure (expired refresh token): prompt user to re-login with message `"Session expired. Run 'kaven auth login' to re-authenticate."`
- [ ] On refresh failure (network error): use existing token if not yet expired, warn user
- [ ] `getValidToken()` method: returns valid access_token or throws if unauthenticated
- [ ] `kaven auth whoami` decodes JWT and displays:
  - Email address
  - GitHub username
  - Current tier (starter/complete/pro)
  - Token expiry time (human-readable)
- [ ] `kaven auth logout` removes `~/.kaven/auth.json` and confirms `"Logged out successfully."`
- [ ] Auth file stores both tokens: `{ access_token, refresh_token, expires_at, user }`
- [ ] File permissions remain `0600` on auth.json
- [ ] `isAuthenticated()` method returns boolean without throwing

### Target Implementation

```typescript
// src/core/AuthService.ts — TARGET
import { MarketplaceClient } from '../infrastructure/MarketplaceClient';

interface StoredAuth {
  access_token: string;
  refresh_token: string;
  expires_at: string; // ISO 8601
  user: {
    email: string;
    githubId: string;
    tier: 'starter' | 'complete' | 'pro' | 'enterprise';
  };
}

interface JwtPayload {
  sub: string;
  email: string;
  githubId: string;
  tier: string;
  exp: number;
  iat: number;
}

export class AuthService {
  private tokenPath: string;
  private cachedAuth: StoredAuth | null = null;

  constructor(private configDir = path.join(os.homedir(), '.kaven')) {
    this.tokenPath = path.join(this.configDir, 'auth.json');
  }

  /**
   * Decode JWT payload without signature verification.
   * CLI should never validate signatures — that's the server's responsibility.
   */
  private decodeJwt(token: string): JwtPayload {
    const parts = token.split('.');
    if (parts.length !== 3) throw new Error('Invalid JWT format');
    const payload = Buffer.from(parts[1], 'base64url').toString('utf-8');
    return JSON.parse(payload);
  }

  /**
   * Returns a valid access token, refreshing if needed.
   * Throws if not authenticated or refresh fails.
   */
  async getValidToken(): Promise<string> {
    const auth = this.loadAuth();
    if (!auth) {
      throw new Error('Not authenticated. Run: kaven auth login');
    }

    const expiresAt = new Date(auth.expires_at).getTime();
    const fiveMinutes = 5 * 60 * 1000;
    const now = Date.now();

    // Token still valid and not expiring soon
    if (now < expiresAt - fiveMinutes) {
      return auth.access_token;
    }

    // Token expiring soon or expired — try refresh
    try {
      const client = new MarketplaceClient(this);
      const refreshed = await client.refreshToken(auth.refresh_token);
      this.saveTokens({
        access_token: refreshed.access_token,
        refresh_token: refreshed.refresh_token,
        expires_at: this.expiryFromJwt(refreshed.access_token),
        user: this.userFromJwt(refreshed.access_token),
      });
      return refreshed.access_token;
    } catch (error) {
      // If token is still valid (just expiring soon), use it
      if (now < expiresAt) {
        console.warn(chalk.yellow('Warning: Token refresh failed. Using existing token.'));
        return auth.access_token;
      }
      throw new Error('Session expired. Run \'kaven auth login\' to re-authenticate.');
    }
  }

  isAuthenticated(): boolean {
    const auth = this.loadAuth();
    return auth !== null;
  }

  saveTokens(auth: StoredAuth): void {
    fs.mkdirSync(this.configDir, { recursive: true });
    fs.writeFileSync(this.tokenPath, JSON.stringify(auth, null, 2));
    fs.chmodSync(this.tokenPath, 0o600);
    this.cachedAuth = auth;
  }

  logout(): void {
    try {
      fs.unlinkSync(this.tokenPath);
    } catch { /* file may not exist */ }
    this.cachedAuth = null;
  }

  getUserInfo(): StoredAuth['user'] | null {
    const auth = this.loadAuth();
    return auth?.user ?? null;
  }

  getExpiryInfo(): { expiresAt: Date; isExpired: boolean; expiresIn: string } | null {
    const auth = this.loadAuth();
    if (!auth) return null;
    const expiresAt = new Date(auth.expires_at);
    const isExpired = Date.now() > expiresAt.getTime();
    const diff = expiresAt.getTime() - Date.now();
    const hours = Math.floor(Math.abs(diff) / (1000 * 60 * 60));
    const minutes = Math.floor((Math.abs(diff) % (1000 * 60 * 60)) / (1000 * 60));
    const expiresIn = isExpired
      ? `expired ${hours}h ${minutes}m ago`
      : `expires in ${hours}h ${minutes}m`;
    return { expiresAt, isExpired, expiresIn };
  }

  private loadAuth(): StoredAuth | null {
    if (this.cachedAuth) return this.cachedAuth;
    try {
      const data = fs.readFileSync(this.tokenPath, 'utf-8');
      this.cachedAuth = JSON.parse(data);
      return this.cachedAuth;
    } catch {
      return null;
    }
  }

  private expiryFromJwt(token: string): string {
    const payload = this.decodeJwt(token);
    return new Date(payload.exp * 1000).toISOString();
  }

  private userFromJwt(token: string): StoredAuth['user'] {
    const payload = this.decodeJwt(token);
    return { email: payload.email, githubId: payload.githubId, tier: payload.tier as any };
  }
}
```
