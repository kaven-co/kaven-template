# C1.5 — Marketplace Install Real

> **Epic:** 2 (CLI) | **Sprint:** C1 | **Agent:** @dev
> **Priority:** P0
> **Depends on:** C1.2 (Real MarketplaceClient), M2.2 (Marketplace — Download Token/Artifact endpoints)

---

## Contexto

The current `kaven marketplace install <slug>` command is mocked — it pretends to download a module and then delegates to the real `ModuleInstaller.install()` which does real marker-based file injection. The download/verification step is entirely fake. This story implements the full real flow: authenticate, fetch module metadata, obtain a download token, download the tar.gz artifact, extract it, and then delegate to the existing (working) `ModuleInstaller`.

The key insight is that `ModuleInstaller` and `MarkerService` are already real and working — we only need to replace the fake download pipeline that feeds modules into them.

## User Story

**As a** developer using kaven-cli,
**I want to** install real modules from the Kaven marketplace with a single command,
**So that** I can add functionality to my project without manual setup.

---

## Acceptance Criteria

- [ ] `kaven marketplace install <slug>` performs the full real install flow:
  1. Verify authentication (auto-refresh token if needed)
  2. `GET /modules/:slug` to verify module exists and get metadata
  3. `POST /download-tokens` to get a time-limited download token
  4. `GET /artifacts/:token` to download the tar.gz artifact
  5. Extract tar.gz to a temp directory
  6. Read `module.json` manifest from extracted files
  7. Delegate to `ModuleInstaller.install()` (existing, working code)
  8. Clean up temp directory on success or failure
- [ ] Progress bar displayed during download using `ora` or `cli-progress`
- [ ] `--version <ver>` flag to install specific version (default: `latest`)
- [ ] Conflict detection: if module markers already present in project, prompt: `"Module 'payments' is already installed. Overwrite? [y/N]"`
- [ ] `--force` flag to skip overwrite confirmation
- [ ] Rollback on failure: if `ModuleInstaller` fails, clean up any partially written files (TransactionalFileSystem handles this)
- [ ] Temp directory cleaned up in `finally` block (even on errors)
- [ ] Success message: `"Module 'payments' v1.2.0 installed successfully."`
- [ ] If not authenticated: show `"Authentication required. Run: kaven auth login"` and exit
- [ ] If license validation fails (403): show tier info and upgrade prompt (delegates to C1.6)
- [ ] Download size shown: `"Downloading payments v1.2.0 (245 KB)..."`
- [ ] SHA-256 checksum verification after download (if provided by API)

### Target Implementation

```typescript
// src/commands/marketplace/install.ts — TARGET
import * as fs from 'fs';
import * as path from 'path';
import * as os from 'os';
import { pipeline } from 'stream/promises';
import * as tar from 'tar';
import { createHash } from 'crypto';
import ora from 'ora';
import chalk from 'chalk';
import { MarketplaceClient } from '../../infrastructure/MarketplaceClient';
import { AuthService } from '../../core/AuthService';
import { ModuleInstaller } from '../../core/ModuleInstaller';
import { MarkerService } from '../../core/MarkerService';
import { AuthenticationError, LicenseRequiredError } from '../../infrastructure/errors';

interface InstallOptions {
  version?: string;
  force?: boolean;
}

export async function installCommand(slug: string, options: InstallOptions) {
  const authService = new AuthService();
  const client = new MarketplaceClient(authService);
  const installer = new ModuleInstaller();
  const markerService = new MarkerService();
  const projectDir = process.cwd();

  // Step 1: Verify authentication
  const spinner = ora('Verifying authentication...').start();
  let token: string;
  try {
    token = await authService.getValidToken();
  } catch {
    spinner.fail('Authentication required. Run: kaven auth login');
    process.exit(1);
  }

  // Step 2: Fetch module metadata
  spinner.text = `Fetching module '${slug}'...`;
  const module = await client.getModule(slug);
  const version = options.version ?? module.version; // latest if not specified

  // Step 3: Check if already installed
  if (!options.force && markerService.isModuleInstalled(slug, projectDir)) {
    spinner.stop();
    const { confirm } = await import('@inquirer/prompts');
    const overwrite = await confirm({
      message: `Module '${slug}' is already installed. Overwrite?`,
      default: false,
    });
    if (!overwrite) {
      console.log(chalk.dim('  Installation cancelled.'));
      return;
    }
    spinner.start();
  }

  // Step 4: Get download token
  spinner.text = 'Requesting download authorization...';
  let downloadToken;
  try {
    downloadToken = await client.createDownloadToken(module.id, version);
  } catch (error) {
    if (error instanceof LicenseRequiredError) {
      spinner.fail(`License required: ${error.requiredTier} tier or higher.`);
      console.log(chalk.yellow(`  Your tier doesn't include this module.`));
      console.log(chalk.dim(`  Run: kaven upgrade ${error.requiredTier}`));
      process.exit(1);
    }
    throw error;
  }

  // Step 5: Download artifact
  const tempDir = fs.mkdtempSync(path.join(os.tmpdir(), 'kaven-install-'));
  const tarPath = path.join(tempDir, `${slug}-${version}.tar.gz`);

  try {
    spinner.text = `Downloading ${slug} v${version}...`;

    const response = await fetch(downloadToken.artifactUrl);
    if (!response.ok) {
      throw new Error(`Download failed: HTTP ${response.status}`);
    }

    const contentLength = response.headers.get('content-length');
    if (contentLength) {
      const sizeKB = Math.round(parseInt(contentLength) / 1024);
      spinner.text = `Downloading ${slug} v${version} (${sizeKB} KB)...`;
    }

    const fileStream = fs.createWriteStream(tarPath);
    await pipeline(response.body as any, fileStream);

    // Step 5b: Verify checksum (if provided)
    if (downloadToken.checksum) {
      spinner.text = 'Verifying checksum...';
      const hash = createHash('sha256');
      const fileBuffer = fs.readFileSync(tarPath);
      hash.update(fileBuffer);
      const computed = hash.digest('hex');
      if (computed !== downloadToken.checksum) {
        throw new Error(
          `Checksum mismatch: expected ${downloadToken.checksum}, got ${computed}`
        );
      }
    }

    // Step 6: Extract tar.gz
    spinner.text = 'Extracting module...';
    const extractDir = path.join(tempDir, 'extracted');
    fs.mkdirSync(extractDir);
    await tar.extract({ file: tarPath, cwd: extractDir });

    // Step 7: Read manifest
    const manifestPath = path.join(extractDir, 'module.json');
    if (!fs.existsSync(manifestPath)) {
      throw new Error('Invalid module package: missing module.json manifest');
    }
    const manifest = JSON.parse(fs.readFileSync(manifestPath, 'utf-8'));

    // Step 8: Delegate to ModuleInstaller (already real and working)
    spinner.text = `Installing ${slug} v${version}...`;
    await installer.install(slug, projectDir, { sourcePath: extractDir, manifest });

    spinner.succeed(
      `Module '${chalk.green(slug)}' v${version} installed successfully.`
    );

    // Post-install hints
    if (manifest.postInstallHint) {
      console.log(chalk.dim(`\n  ${manifest.postInstallHint}`));
    }
  } finally {
    // Step 9: Cleanup temp directory (always)
    try {
      fs.rmSync(tempDir, { recursive: true, force: true });
    } catch {
      // Best effort cleanup
    }
  }
}
```

### Commander.js Registration

```typescript
marketplace
  .command('install <slug>')
  .description('Install a module from the Kaven marketplace')
  .option('-v, --version <version>', 'Specific version to install')
  .option('-f, --force', 'Overwrite existing module without confirmation')
  .action(async (slug, opts) => {
    await installCommand(slug, opts);
  });
```
