# C1.4 — Marketplace List Real

> **Epic:** 2 (CLI) | **Sprint:** C1 | **Agent:** @dev
> **Priority:** P0
> **Depends on:** C1.2 (Real MarketplaceClient)

---

## Contexto

The current `kaven marketplace list` command displays 3 hardcoded modules from mock data in `MarketplaceClient`. With C1.2 providing a real HTTP client, this story wires the list command to the actual marketplace API. The output should be a formatted table with filtering, sorting, and pagination options, replacing the mock entirely.

## User Story

**As a** developer using kaven-cli,
**I want to** browse real modules available in the Kaven marketplace,
**So that** I can discover and choose modules to install in my project.

---

## Acceptance Criteria

- [ ] `kaven marketplace list` calls `GET /modules` via real `MarketplaceClient`
- [ ] Output displayed as formatted table with columns: Slug, Name, Version, Tier, Installs
- [ ] Table uses `cli-table3` or similar for aligned column output
- [ ] `--category <cat>` flag filters by category (e.g., `--category payments`)
- [ ] `--sort <field>` flag sorts results: `newest` (default), `popular`, `name`
- [ ] `--page <n>` flag for pagination (default: 1)
- [ ] `--limit <n>` flag for results per page (default: 20, max: 100)
- [ ] Color coding for tier column:
  - Green: `starter` (free tier modules)
  - Yellow: `complete` (mid-tier)
  - Blue/Magenta: `pro` (premium)
- [ ] Pagination footer: `"Showing 1-20 of 156 modules (page 1/8)"`
- [ ] If not authenticated: show warning `"Not logged in. Some modules may require authentication. Run: kaven auth login"` but still show public modules
- [ ] Empty results: show `"No modules found matching your criteria."`
- [ ] Network error: show `"Could not reach marketplace. Check your connection."` with suggestion to try `--offline` (future C2.6)
- [ ] `--json` flag outputs raw JSON for scripting

### Target Implementation

```typescript
// src/commands/marketplace/list.ts — TARGET
import Table from 'cli-table3';
import chalk from 'chalk';
import { MarketplaceClient } from '../../infrastructure/MarketplaceClient';
import { AuthService } from '../../core/AuthService';
import { AuthenticationError, NetworkError } from '../../infrastructure/errors';

interface ListOptions {
  category?: string;
  sort?: 'newest' | 'popular' | 'name';
  page?: number;
  limit?: number;
  json?: boolean;
}

export async function listCommand(options: ListOptions) {
  const authService = new AuthService();
  const client = new MarketplaceClient(authService);

  // Warn if not authenticated (but don't block)
  if (!authService.isAuthenticated()) {
    console.log(chalk.yellow(
      '  Not logged in. Some modules may require authentication.\n' +
      '  Run: kaven auth login\n'
    ));
  }

  try {
    const result = await client.listModules({
      category: options.category,
      sort: options.sort ?? 'newest',
      page: options.page ?? 1,
      limit: Math.min(options.limit ?? 20, 100),
    });

    // JSON output for scripting
    if (options.json) {
      console.log(JSON.stringify(result, null, 2));
      return;
    }

    if (result.data.length === 0) {
      console.log(chalk.dim('  No modules found matching your criteria.'));
      return;
    }

    // Table output
    const table = new Table({
      head: [
        chalk.bold('Slug'),
        chalk.bold('Name'),
        chalk.bold('Version'),
        chalk.bold('Tier'),
        chalk.bold('Installs'),
      ],
      colWidths: [22, 30, 10, 12, 10],
      style: { head: [], border: ['dim'] },
    });

    for (const mod of result.data) {
      table.push([
        mod.slug,
        mod.name,
        `v${mod.version}`,
        colorTier(mod.tier),
        formatInstalls(mod.installs),
      ]);
    }

    console.log(table.toString());

    // Pagination footer
    const { page, total, totalPages } = result.pagination;
    const start = (page - 1) * result.pagination.limit + 1;
    const end = Math.min(page * result.pagination.limit, total);
    console.log(chalk.dim(`\n  Showing ${start}-${end} of ${total} modules (page ${page}/${totalPages})`));

    if (page < totalPages) {
      console.log(chalk.dim(`  Next page: kaven marketplace list --page ${page + 1}`));
    }
  } catch (error) {
    if (error instanceof NetworkError) {
      console.error(chalk.red('  Could not reach marketplace. Check your connection.'));
      process.exit(1);
    }
    throw error;
  }
}

function colorTier(tier: string): string {
  switch (tier) {
    case 'starter': return chalk.green(tier);
    case 'complete': return chalk.yellow(tier);
    case 'pro': return chalk.magenta(tier);
    default: return tier;
  }
}

function formatInstalls(count: number): string {
  if (count >= 1000) return `${(count / 1000).toFixed(1)}k`;
  return String(count);
}
```

### Commander.js Registration

```typescript
// In parent command registration
marketplace
  .command('list')
  .description('List available modules from the Kaven marketplace')
  .option('-c, --category <category>', 'Filter by category')
  .option('-s, --sort <sort>', 'Sort by: newest, popular, name', 'newest')
  .option('-p, --page <page>', 'Page number', '1')
  .option('-l, --limit <limit>', 'Results per page (max 100)', '20')
  .option('--json', 'Output raw JSON')
  .action(async (opts) => {
    await listCommand({
      ...opts,
      page: parseInt(opts.page, 10),
      limit: parseInt(opts.limit, 10),
    });
  });
```
