# C1.2 — Real MarketplaceClient

> **Epic:** 2 (CLI) | **Sprint:** C1 | **Agent:** @dev
> **Priority:** P0
> **Depends on:** M1.6 (Marketplace API — Module Listing endpoints)

---

## Contexto

The current `src/infrastructure/MarketplaceClient.ts` is entirely mocked — every method returns hardcoded data (3 fake modules: "payments", "notifications", "analytics"). No HTTP calls are made. The real kaven-marketplace API is being built in parallel (Epic M1/M2), and this story replaces all mocks with a production-ready HTTP client that communicates with the marketplace REST API.

This is a foundational story — nearly every other C1 story depends on `MarketplaceClient` being real. It must handle authentication headers, retries, timeouts, and proper error mapping.

## User Story

**As a** kaven-cli developer,
**I want to** replace the mock MarketplaceClient with a real HTTP client,
**So that** all marketplace commands (list, install, publish) communicate with the actual kaven-marketplace API.

## Acceptance Criteria

- [ ] `MarketplaceClient` uses `undici` (Node.js built-in) or native `fetch` for HTTP calls
- [ ] Base URL configurable via `~/.kaven/config.json` field `apiUrl` or `KAVEN_API_URL` env var
- [ ] Default base URL: `https://api.kaven.site` (production)
- [ ] Dev override: `http://localhost:3000` when `KAVEN_API_URL` is set
- [ ] All methods implemented with real HTTP calls:
  - `listModules(filters?)` → `GET /modules`
  - `getModule(slug)` → `GET /modules/:slug`
  - `getManifest(slug, version)` → `GET /modules/:slug/versions/:version/manifest`
  - `createDownloadToken(moduleId, releaseId)` → `POST /download-tokens`
  - `requestDeviceCode()` → `POST /auth/device-code`
  - `pollDeviceToken(deviceCode)` → `POST /auth/token`
  - `refreshToken(refreshToken)` → `POST /auth/refresh`
- [ ] JWT `Authorization: Bearer <token>` header auto-attached from `AuthService` on authenticated endpoints
- [ ] Retry logic: 3 retries with exponential backoff (1s, 2s, 4s) for 5xx errors and network timeouts
- [ ] No retries on 4xx errors (client errors are not transient)
- [ ] Request timeout: 30 seconds default, configurable per-method
- [ ] Error handling maps HTTP status codes to typed errors:
  - `401` → `AuthenticationError` (prompt re-login)
  - `403` → `LicenseRequiredError` (show tier info)
  - `404` → `NotFoundError`
  - `429` → `RateLimitError` (show retry-after header value)
  - `5xx` → `ServerError` (after retries exhausted)
  - Network error → `NetworkError`
- [ ] Response parsing: JSON with type validation
- [ ] Request/response logging in debug mode (`KAVEN_DEBUG=1`)

### Target Implementation

```typescript
// src/infrastructure/MarketplaceClient.ts — TARGET
import { AuthService } from '../core/AuthService';

interface MarketplaceConfig {
  baseUrl: string;
  timeout: number;
  maxRetries: number;
}

export class MarketplaceClient {
  private config: MarketplaceConfig;
  private authService: AuthService;

  constructor(authService: AuthService, config?: Partial<MarketplaceConfig>) {
    this.authService = authService;
    this.config = {
      baseUrl: process.env.KAVEN_API_URL
        ?? this.loadConfigUrl()
        ?? 'https://api.kaven.site',
      timeout: config?.timeout ?? 30_000,
      maxRetries: config?.maxRetries ?? 3,
    };
  }

  private loadConfigUrl(): string | undefined {
    try {
      const configPath = path.join(os.homedir(), '.kaven', 'config.json');
      const config = JSON.parse(fs.readFileSync(configPath, 'utf-8'));
      return config.apiUrl;
    } catch {
      return undefined;
    }
  }

  private async request<T>(
    method: string,
    path: string,
    options?: { body?: unknown; authenticated?: boolean; timeout?: number }
  ): Promise<T> {
    const url = `${this.config.baseUrl}${path}`;
    const headers: Record<string, string> = {
      'Content-Type': 'application/json',
      'User-Agent': `kaven-cli/${CLI_VERSION}`,
    };

    if (options?.authenticated !== false) {
      const token = await this.authService.getValidToken();
      if (token) headers['Authorization'] = `Bearer ${token}`;
    }

    let lastError: Error | undefined;

    for (let attempt = 0; attempt <= this.config.maxRetries; attempt++) {
      try {
        const controller = new AbortController();
        const timeoutId = setTimeout(
          () => controller.abort(),
          options?.timeout ?? this.config.timeout
        );

        const response = await fetch(url, {
          method,
          headers,
          body: options?.body ? JSON.stringify(options.body) : undefined,
          signal: controller.signal,
        });

        clearTimeout(timeoutId);

        if (response.ok) {
          return await response.json() as T;
        }

        // Don't retry client errors
        if (response.status < 500) {
          throw this.mapHttpError(response.status, await response.json());
        }

        // Retry on 5xx
        lastError = new ServerError(`Server error: ${response.status}`);

        if (attempt < this.config.maxRetries) {
          const delay = Math.pow(2, attempt) * 1000; // 1s, 2s, 4s
          this.debug(`Retry ${attempt + 1}/${this.config.maxRetries} after ${delay}ms`);
          await sleep(delay);
        }
      } catch (error) {
        if (error instanceof MarketplaceError) throw error;
        if (error.name === 'AbortError') {
          lastError = new NetworkError('Request timeout');
        } else {
          lastError = new NetworkError(`Network error: ${error.message}`);
        }

        if (attempt < this.config.maxRetries) {
          const delay = Math.pow(2, attempt) * 1000;
          await sleep(delay);
        }
      }
    }

    throw lastError ?? new NetworkError('Request failed after retries');
  }

  private mapHttpError(status: number, body: any): MarketplaceError {
    switch (status) {
      case 401: return new AuthenticationError('Authentication required. Run: kaven auth login');
      case 403: return new LicenseRequiredError(body.requiredTier ?? 'unknown', body.message);
      case 404: return new NotFoundError(body.message ?? 'Resource not found');
      case 429: return new RateLimitError(body.retryAfter ?? 60);
      default:  return new MarketplaceError(`HTTP ${status}: ${body.message ?? 'Unknown error'}`);
    }
  }

  // --- Public API Methods ---

  async listModules(filters?: ModuleFilters): Promise<PaginatedResponse<Module>> {
    const params = new URLSearchParams();
    if (filters?.category) params.set('category', filters.category);
    if (filters?.sort) params.set('sort', filters.sort);
    if (filters?.page) params.set('page', String(filters.page));
    if (filters?.limit) params.set('limit', String(filters.limit));
    const query = params.toString() ? `?${params}` : '';
    return this.request('GET', `/modules${query}`);
  }

  async getModule(slug: string): Promise<Module> {
    return this.request('GET', `/modules/${slug}`);
  }

  async getManifest(slug: string, version: string): Promise<ModuleManifest> {
    return this.request('GET', `/modules/${slug}/versions/${version}/manifest`);
  }

  async createDownloadToken(moduleId: string, releaseId: string): Promise<DownloadToken> {
    return this.request('POST', '/download-tokens', {
      body: { moduleId, releaseId },
      authenticated: true,
    });
  }

  // Auth methods (unauthenticated)
  async requestDeviceCode(): Promise<DeviceCodeResponse> {
    return this.request('POST', '/auth/device-code', {
      body: { client_id: 'kaven-cli' },
      authenticated: false,
    });
  }

  async pollDeviceToken(deviceCode: string): Promise<TokenPollResponse> {
    return this.request('POST', '/auth/token', {
      body: { device_code: deviceCode, grant_type: 'urn:ietf:params:oauth:grant-type:device_code' },
      authenticated: false,
    });
  }

  async refreshToken(refreshToken: string): Promise<TokenResponse> {
    return this.request('POST', '/auth/refresh', {
      body: { refresh_token: refreshToken },
      authenticated: false,
    });
  }

  private debug(msg: string) {
    if (process.env.KAVEN_DEBUG === '1') {
      console.error(`[MarketplaceClient] ${msg}`);
    }
  }
}
```

### Error Class Hierarchy

```typescript
// src/infrastructure/errors.ts
export class MarketplaceError extends Error {
  constructor(message: string) { super(message); this.name = 'MarketplaceError'; }
}
export class AuthenticationError extends MarketplaceError {
  constructor(message: string) { super(message); this.name = 'AuthenticationError'; }
}
export class LicenseRequiredError extends MarketplaceError {
  constructor(public requiredTier: string, message: string) {
    super(message); this.name = 'LicenseRequiredError';
  }
}
export class NotFoundError extends MarketplaceError {
  constructor(message: string) { super(message); this.name = 'NotFoundError'; }
}
export class RateLimitError extends MarketplaceError {
  constructor(public retryAfter: number) {
    super(`Rate limited. Try again in ${retryAfter}s`);
    this.name = 'RateLimitError';
  }
}
export class ServerError extends MarketplaceError {
  constructor(message: string) { super(message); this.name = 'ServerError'; }
}
export class NetworkError extends MarketplaceError {
  constructor(message: string) { super(message); this.name = 'NetworkError'; }
}
```
