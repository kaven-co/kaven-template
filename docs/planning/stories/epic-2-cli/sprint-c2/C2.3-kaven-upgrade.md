# C2.3 — `kaven upgrade` Command

> **Epic:** 2 (CLI) | **Sprint:** C2 | **Agent:** @dev
> **Priority:** P1
> **Depends on:** M2.5 (Marketplace — Checkout Session endpoint)

---

## Contexto

The pricing page in the Kaven framework (Tenant App) has an "Upgrade" button that is not yet wired to a real payment gateway (this is the last remaining tech debt item FE-M2). The CLI-side `kaven upgrade` command provides an alternative upgrade path: it shows the user's current tier, displays a comparison of available tiers, and initiates a Paddle checkout session in the browser. This command is particularly useful because many developers prefer managing their subscriptions from the terminal.

Paddle is used as the payment gateway for marketplace licenses (as documented in the architecture: "Paddle for marketplace licenses"). The CLI creates a checkout session via the marketplace API, opens the checkout URL in the browser, and either polls for completion or relies on a local callback.

## User Story

**As a** developer using kaven-cli,
**I want to** upgrade my license tier directly from the CLI,
**So that** I can access more modules without leaving my terminal workflow.

### Implementation

```typescript
// src/commands/upgrade/upgrade.ts
import open from 'open';
import ora from 'ora';
import chalk from 'chalk';
import { MarketplaceClient } from '../../infrastructure/MarketplaceClient';
import { AuthService } from '../../core/AuthService';
import { LicenseService } from '../../core/LicenseService';
import { showTierComparison } from '../license/tier-table';

const TIER_ORDER = ['starter', 'complete', 'pro', 'enterprise'];

interface UpgradeOptions {
  noBrowser?: boolean;
}

export async function upgradeCommand(targetTier?: string, options?: UpgradeOptions) {
  const authService = new AuthService();
  const client = new MarketplaceClient(authService);
  const licenseService = new LicenseService();

  // Verify auth
  if (!authService.isAuthenticated()) {
    console.error(chalk.red('Login required. Run: kaven auth login'));
    process.exit(1);
  }

  // Get current status
  const user = authService.getUserInfo();
  const currentTier = user?.tier ?? 'starter';

  // No target tier — show info only
  if (!targetTier) {
    console.log(chalk.bold('\n  Current Plan\n'));
    console.log(`  Tier:     ${chalk.green(currentTier)}`);

    const expiry = authService.getExpiryInfo();
    if (expiry) {
      console.log(`  License:  ${expiry.expiresIn}`);
    }

    showTierComparison(currentTier, '');
    console.log(chalk.dim('  To upgrade: kaven upgrade <tier>'));
    console.log(chalk.dim('  Example:    kaven upgrade complete\n'));
    return;
  }

  // Validate target tier
  const normalizedTarget = targetTier.toLowerCase();
  if (!TIER_ORDER.includes(normalizedTarget)) {
    console.error(chalk.red(`Unknown tier '${targetTier}'. Available: ${TIER_ORDER.join(', ')}`));
    process.exit(1);
  }

  const currentIdx = TIER_ORDER.indexOf(currentTier);
  const targetIdx = TIER_ORDER.indexOf(normalizedTarget);

  if (currentIdx === targetIdx) {
    console.log(chalk.yellow(`You're already on the ${currentTier} plan.`));
    return;
  }

  if (targetIdx < currentIdx) {
    console.log(chalk.yellow('Downgrades are handled at https://kaven.sh/account'));
    return;
  }

  if (normalizedTarget === 'enterprise') {
    console.log(chalk.bold('\n  Enterprise plans require custom pricing.'));
    console.log(chalk.dim('  Contact: sales@kaven.sh\n'));
    return;
  }

  // Show what they're getting
  showTierComparison(currentTier, normalizedTarget);

  // Create checkout session
  const spinner = ora('Creating checkout session...').start();
  const { checkoutUrl, sessionId } = await client.createCheckoutSession({
    targetTier: normalizedTarget,
    currentTier,
  });
  spinner.succeed('Checkout session created');

  // Open browser
  if (options?.noBrowser) {
    console.log(chalk.bold('\n  Complete your payment at:'));
    console.log(chalk.underline(`  ${checkoutUrl}\n`));
  } else {
    try {
      await open(checkoutUrl);
      console.log(chalk.dim('\n  Browser opened for payment.'));
    } catch {
      console.log(chalk.bold('\n  Open this URL to complete payment:'));
      console.log(chalk.underline(`  ${checkoutUrl}\n`));
    }
  }

  // Poll for completion
  const pollSpinner = ora('Waiting for payment confirmation...').start();
  const maxWait = 10 * 60 * 1000; // 10 minutes
  const pollInterval = 5 * 1000;  // 5 seconds
  const deadline = Date.now() + maxWait;

  while (Date.now() < deadline) {
    await sleep(pollInterval);

    try {
      const status = await client.getCheckoutStatus(sessionId);

      if (status.state === 'completed') {
        pollSpinner.succeed('Payment confirmed!');

        // Update local cache
        const newModuleCount = status.additionalModules ?? 0;
        console.log(chalk.green.bold(
          `\n  Upgraded to ${normalizedTarget}! You now have access to ${newModuleCount} additional modules.\n`
        ));

        // Invalidate license cache
        licenseService.clearCache();
        return;
      }

      if (status.state === 'cancelled') {
        pollSpinner.info('Payment cancelled. Your plan was not changed.');
        return;
      }

      // 'pending' — continue polling
      const remaining = Math.ceil((deadline - Date.now()) / 1000);
      const mm = Math.floor(remaining / 60);
      const ss = String(remaining % 60).padStart(2, '0');
      pollSpinner.text = `Waiting for payment confirmation... (${mm}:${ss} remaining)`;
    } catch {
      // Network error during poll — continue silently
    }
  }

  pollSpinner.warn(
    'Payment not completed within 10 minutes.\n' +
    '  Check your email for confirmation or try again.'
  );
}

function sleep(ms: number): Promise<void> {
  return new Promise(resolve => setTimeout(resolve, ms));
}
```

### MarketplaceClient New Methods

```typescript
// Added to MarketplaceClient
async createCheckoutSession(params: {
  targetTier: string;
  currentTier: string;
}): Promise<{ checkoutUrl: string; sessionId: string }> {
  return this.request('POST', '/checkout/sessions', {
    body: params,
    authenticated: true,
  });
}

async getCheckoutStatus(sessionId: string): Promise<{
  state: 'pending' | 'completed' | 'cancelled';
  additionalModules?: number;
}> {
  return this.request('GET', `/checkout/sessions/${sessionId}/status`, {
    authenticated: true,
  });
}
```
