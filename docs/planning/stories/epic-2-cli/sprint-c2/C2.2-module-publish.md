# C2.2 — `kaven module publish`

> **Epic:** 2 (CLI) | **Sprint:** C2 | **Agent:** @dev
> **Priority:** P1
> **Depends on:** M2.3 (Marketplace — Presigned Upload & Release Creation endpoints)

---

## Contexto

For the Kaven marketplace ecosystem to grow, module authors need a way to publish their modules. The `kaven module publish` command packages a module directory into a signed tar.gz artifact, uploads it to S3 via a presigned URL, and creates a release record in the marketplace API. This is the counterpart to `kaven marketplace install` — one publishes, the other consumes.

Security is critical: every artifact is signed with Ed25519 (the author's key) and verified with SHA-256 checksums. The marketplace API validates ownership and the release integrity before making it available.

## User Story

**As a** module author,
**I want to** publish my module to the Kaven marketplace from the CLI,
**So that** other developers can discover and install my module.

## Acceptance Criteria

- [ ] `kaven module publish` runs from a directory containing `module.json`
- [ ] If no `module.json` found: error `"No module.json found. Are you in a module directory?"`
- [ ] Validates `module.json` against Zod schema before proceeding
- [ ] Validation checks: slug (kebab-case), version (semver), name, description, files array
- [ ] If validation fails: show all errors with line references
- [ ] Creates tar.gz of the module directory (excluding `node_modules`, `.git`, `dist`)
- [ ] Generates Ed25519 signature of the tar.gz using author's private key
- [ ] Private key location: `~/.kaven/keys/signing-key.pem`
- [ ] If no signing key exists: prompt to generate one with `kaven auth generate-key`
- [ ] Calculates SHA-256 checksum of the tar.gz
- [ ] Calls `POST /modules/:slug/releases/upload-url` to get presigned S3 URL
- [ ] Uploads tar.gz to the presigned S3 URL via PUT
- [ ] Progress bar during upload (shows bytes uploaded / total)
- [ ] After upload: calls `POST /modules/:slug/releases` with:
  - version, checksum, signature, artifact size, changelog
- [ ] Requires authentication (JWT)
- [ ] Requires module ownership or ADMIN role
- [ ] If module doesn't exist yet: prompt `"Module '<slug>' doesn't exist. Create it? [Y/n]"`
- [ ] On success: `"Published <slug> v<version> successfully. Available at https://kaven.sh/modules/<slug>"`
- [ ] `--dry-run` flag: validate and package but don't upload
- [ ] `--changelog <text>` flag: attach changelog to release (or read from CHANGELOG.md)

## Technical Notes

### Module Manifest Zod Schema

```typescript
// src/core/manifest-schema.ts
import { z } from 'zod';

export const moduleManifestSchema = z.object({
  name: z.string().min(1).max(100),
  slug: z.string().regex(/^[a-z0-9-]+$/, 'Must be kebab-case'),
  version: z.string().regex(/^\d+\.\d+\.\d+(-[\w.]+)?$/, 'Must be valid semver'),
  description: z.string().min(10).max(500),
  author: z.object({
    name: z.string(),
    email: z.string().email().optional(),
    githubId: z.string().optional(),
  }),
  tier: z.enum(['starter', 'complete', 'pro']),
  category: z.string(),
  files: z.array(z.object({
    source: z.string(),
    target: z.string(),
    action: z.enum(['create', 'inject', 'append']),
    anchor: z.string().optional(),
  })).min(1),
  dependencies: z.record(z.string()).optional(),
  peerDependencies: z.record(z.string()).optional(),
  postInstall: z.array(z.string()).optional(),
  preRemove: z.array(z.string()).optional(),
  env: z.array(z.object({
    name: z.string(),
    description: z.string(),
    default: z.string().optional(),
    sensitive: z.boolean().optional(),
    required: z.boolean().optional(),
  })).optional(),
  kavenVersion: z.string().optional(), // Minimum compatible framework version
});

export type ModuleManifest = z.infer<typeof moduleManifestSchema>;
```

### Publish Implementation

```typescript
// src/commands/module/publish.ts
import * as fs from 'fs';
import * as path from 'path';
import * as crypto from 'crypto';
import * as tar from 'tar';
import chalk from 'chalk';
import ora from 'ora';
import { MarketplaceClient } from '../../infrastructure/MarketplaceClient';
import { AuthService } from '../../core/AuthService';
import { moduleManifestSchema } from '../../core/manifest-schema';

interface PublishOptions {
  dryRun?: boolean;
  changelog?: string;
}

export async function publishCommand(options: PublishOptions) {
  const authService = new AuthService();
  const client = new MarketplaceClient(authService);

  // Step 1: Find and validate module.json
  const manifestPath = path.join(process.cwd(), 'module.json');
  if (!fs.existsSync(manifestPath)) {
    console.error(chalk.red('No module.json found. Are you in a module directory?'));
    process.exit(1);
  }

  const rawManifest = JSON.parse(fs.readFileSync(manifestPath, 'utf-8'));
  const validation = moduleManifestSchema.safeParse(rawManifest);

  if (!validation.success) {
    console.error(chalk.red('\n  module.json validation failed:\n'));
    for (const err of validation.error.errors) {
      console.error(chalk.red(`    ${err.path.join('.')}: ${err.message}`));
    }
    process.exit(1);
  }

  const manifest = validation.data;
  console.log(chalk.bold(`\n  Publishing ${manifest.slug} v${manifest.version}\n`));

  // Step 2: Verify auth
  const spinner = ora('Verifying authentication...').start();
  try {
    await authService.getValidToken();
    spinner.succeed('Authenticated');
  } catch {
    spinner.fail('Authentication required. Run: kaven auth login');
    process.exit(1);
  }

  // Step 3: Create tar.gz
  spinner.start('Packaging module...');
  const tarPath = path.join(
    os.tmpdir(),
    `${manifest.slug}-${manifest.version}.tar.gz`
  );

  try {
    await tar.create(
      {
        gzip: true,
        file: tarPath,
        cwd: process.cwd(),
        filter: (filePath) => {
          // Exclude unwanted directories
          const excluded = ['node_modules', '.git', 'dist', '.DS_Store'];
          return !excluded.some(ex => filePath.includes(ex));
        },
      },
      ['.']
    );

    const stats = fs.statSync(tarPath);
    const sizeKB = Math.round(stats.size / 1024);
    spinner.succeed(`Package created (${sizeKB} KB)`);
  } catch (error) {
    spinner.fail('Failed to create package');
    throw error;
  }

  // Step 4: Generate checksum
  spinner.start('Generating checksum...');
  const fileBuffer = fs.readFileSync(tarPath);
  const checksum = crypto.createHash('sha256').update(fileBuffer).digest('hex');
  spinner.succeed(`Checksum: ${checksum.substring(0, 16)}...`);

  // Step 5: Sign artifact
  spinner.start('Signing artifact...');
  const keyPath = path.join(os.homedir(), '.kaven', 'keys', 'signing-key.pem');

  if (!fs.existsSync(keyPath)) {
    spinner.fail('No signing key found');
    console.log(chalk.yellow('\n  Generate a signing key first:'));
    console.log(chalk.dim('    kaven auth generate-key\n'));
    process.exit(1);
  }

  const privateKey = fs.readFileSync(keyPath, 'utf-8');
  const signature = crypto.sign('sha256', fileBuffer, {
    key: privateKey,
    padding: crypto.constants.RSA_PKCS1_PSS_PADDING,
  }).toString('base64');
  spinner.succeed('Artifact signed');

  // Dry run stops here
  if (options.dryRun) {
    console.log(chalk.green('\n  Dry run complete. Package is valid and signed.'));
    console.log(chalk.dim(`  Package: ${tarPath}`));
    console.log(chalk.dim(`  Checksum: ${checksum}`));
    fs.unlinkSync(tarPath);
    return;
  }

  // Step 6: Get presigned upload URL
  spinner.start('Requesting upload URL...');
  const { uploadUrl, artifactId } = await client.getUploadUrl(
    manifest.slug,
    manifest.version,
    fileBuffer.length
  );
  spinner.succeed('Upload URL obtained');

  // Step 7: Upload to S3
  spinner.start(`Uploading ${manifest.slug} v${manifest.version}...`);
  const uploadResponse = await fetch(uploadUrl, {
    method: 'PUT',
    headers: {
      'Content-Type': 'application/gzip',
      'Content-Length': String(fileBuffer.length),
    },
    body: fileBuffer,
  });

  if (!uploadResponse.ok) {
    spinner.fail('Upload failed');
    throw new Error(`Upload failed: HTTP ${uploadResponse.status}`);
  }
  spinner.succeed('Artifact uploaded');

  // Step 8: Create release
  spinner.start('Creating release...');
  const changelog = options.changelog ?? readChangelogSection(manifest.version);

  await client.createRelease(manifest.slug, {
    version: manifest.version,
    checksum,
    signature,
    artifactId,
    artifactSize: fileBuffer.length,
    changelog,
    manifest: rawManifest,
  });

  spinner.succeed('Release created');

  // Cleanup
  fs.unlinkSync(tarPath);

  // Success
  console.log(chalk.green.bold(
    `\n  Published ${manifest.slug} v${manifest.version} successfully!`
  ));
  console.log(chalk.dim(
    `  Available at: https://kaven.sh/modules/${manifest.slug}\n`
  ));
}

function readChangelogSection(version: string): string | undefined {
  const changelogPath = path.join(process.cwd(), 'CHANGELOG.md');
  if (!fs.existsSync(changelogPath)) return undefined;

  const content = fs.readFileSync(changelogPath, 'utf-8');
  const versionHeader = `## [${version}]`;
  const startIdx = content.indexOf(versionHeader);
  if (startIdx === -1) return undefined;

  const nextHeader = content.indexOf('\n## ', startIdx + 1);
  return content.substring(startIdx, nextHeader === -1 ? undefined : nextHeader).trim();
}
```

// Added to MarketplaceClient
async getUploadUrl(slug: string, version: string, size: number): Promise<UploadUrlResponse> {
  return this.request('POST', `/modules/${slug}/releases/upload-url`, {
    body: { version, size },
    authenticated: true,
  });
}

async createRelease(slug: string, release: CreateReleasePayload): Promise<Release> {
  return this.request('POST', `/modules/${slug}/releases`, {

    authenticated: true,
  });
