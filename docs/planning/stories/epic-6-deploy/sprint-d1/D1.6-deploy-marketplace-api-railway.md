**Service:** `kaven-marketplace`
**Region:** us-east1 (consistente com kaven-api)
**Revision:** `kaven-marketplace-00024-t6t`

**Deploy via gcloud:**
```bash
gcloud run deploy kaven-marketplace \
  --source . \
  --project kaven-488720 \
  --region us-east1 \
  --allow-unauthenticated
```
