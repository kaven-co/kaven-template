# S1.3 — Features Showcase

> **Epic:** 3 (Site — Landing Page MVP) | **Sprint:** S1 | **Agent:** @dev
> **Priority:** P1
> **Depends on:** None

---

## Contexto

O Kaven framework tem 22+ features organizadas em modulos, com metricas impressionantes (959 testes, 33 modelos IDOR, 42+ composite indexes, 76+ componentes de design system). O features showcase precisa transformar essas metricas tecnicas em valor percebido para o desenvolvedor. A pagina serve dois propositos: (1) convencer o visitante de que o Kaven e robusto e completo, e (2) mostrar exatamente o que cada tier inclui para facilitar a decisao de compra.

O diferencial visual sera a combinacao de feature grid categorizado com animacoes de contadores numericos que destacam as metricas-chave e expandable cards com snippets de codigo real do framework.

---

## User Story

**Como** desenvolvedor avaliando o Kaven,
**Quero** ver todas as features organizadas por categoria com detalhes tecnicos e metricas,
**Para que** eu entenda a profundidade do framework e confie que ele resolve meus problemas.

## Technical Notes

- **Route**: `app/features/page.tsx`
- **Animation library**: Framer Motion for complex animations (card expand, counters, fade-in). Install as dependency.
- **Counter animation**: Custom hook `useCountUp(target, duration, startOnView)` using `requestAnimationFrame` and Intersection Observer API.
- **Feature data**: `lib/features-data.ts` — typed array of categories with features:
  ```typescript
  interface Feature {
    id: string;
    title: string;
    description: string;
    detailedDescription: string;
    codeSnippet?: string;
    tiers: ('starter' | 'complete' | 'pro')[];
    icon: string;
    docsLink?: string;
  }
  interface FeatureCategory {
    id: string;
    name: string;
    icon: string;
    color: string;
    features: Feature[];
  }
  ```
- **Code snippets**: Use Shiki for syntax highlighting in expanded cards. Lazy-load Shiki to avoid bundle bloat.
- **Performance**: Feature grid uses CSS Grid. Images/icons are SVG inline or sprite. No heavy images.
- **File structure**:
  ```
  app/features/
  └── page.tsx
  components/features/
  ├── FeatureGrid.tsx          # Main grid with category filter
  ├── FeatureCard.tsx          # Individual feature card (expandable)
  ├── FeatureCategory.tsx      # Category header with icon
  ├── StatsCounter.tsx         # Animated counter section
  ├── CodePreview.tsx          # Syntax-highlighted code in expanded card
  └── CategoryFilter.tsx       # Filter pills/tabs
  lib/
  ├── features-data.ts         # All feature definitions
  └── hooks/useCountUp.ts      # Counter animation hook
  ```
