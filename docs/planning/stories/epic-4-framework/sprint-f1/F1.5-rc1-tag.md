## Contexto

Apos completar o upgrade flow (F1.1), webhook handler (F1.2), usage limit enforcement (F1.3), e payments module package (F1.4), o kaven-framework esta pronto para a tag v1.0.0-rc1. Este e o marco mais importante do projeto: a primeira release candidate que sera usada por early adopters e servira como base para o marketplace e o kaven-site.

O RC1 precisa de validacao rigorosa: todos os testes passando, CI verde, CHANGELOG gerado, versoes atualizadas, tag GPG assinada, e smoke test completo do fluxo de desenvolvimento (kaven init → modulo add → dev server → todas features funcionando).

## Technical Notes

- **CHANGELOG generation**: Use `conventional-changelog-cli` or manual generation from git log:
  ```bash
  # Generate from conventional commits
  npx conventional-changelog -p angular -i CHANGELOG.md -s -r 0

  # Or manual approach
  git log --oneline --no-merges v0.0.0..HEAD | sort
  ```
  Manual curation recommended for RC1 since it's the first release — automated tools may miss context.

- **Version bump script** (if Turborepo workspace):
  ```bash
  # Bump all workspaces
  npm version 1.0.0-rc1 --workspaces --no-git-tag-version
  # Bump root
  npm version 1.0.0-rc1 --no-git-tag-version
  ```

- **GPG tag signing**: Ensure GPG key is configured:
  ```bash
  git config --global user.signingkey YOUR_GPG_KEY_ID
  git tag -s v1.0.0-rc1 -m "Release v1.0.0-rc1 — First release candidate"
  git push origin v1.0.0-rc1
  ```

- **GitHub Release via CLI**:
  ```bash
  gh release create v1.0.0-rc1 \
    --title "v1.0.0-rc1 — Release Candidate 1" \
    --notes-file RELEASE_NOTES.md \
    --prerelease \
    CHANGELOG.md
  ```

- **Smoke test checklist file**: Create `scripts/smoke-test.sh` that automates the smoke test steps (or at minimum documents them in a runnable format).

- **Pre-tag checklist**:
  ```
  [ ] All F1.x PRs merged to main
  [ ] main branch pulled locally
  [ ] npm ci (clean install)
  [ ] npm run lint (0 errors)
  [ ] npm run typecheck (0 errors)
  [ ] npm run test (959+ pass, 0 fail)
  [ ] npm run build (all apps)
  [ ] npm audit (no critical)
  [ ] CHANGELOG.md reviewed and committed

  [ ] CLAUDE.md updated and committed
  [ ] Git tag created and pushed
  [ ] GitHub Release published
  [ ] Smoke test passed
  ```
