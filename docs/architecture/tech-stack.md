# Tech Stack — Kaven Framework

> Generated by `kaven init` — Kaven Framework v1.1.0-alpha (Prisma 7 Consolidation)

## Backend
- **Runtime:** Node.js 20 LTS
- **Framework:** Fastify 5.8.5 (REST API, port 8000)
- **ORM:** Prisma 7.0.0 (Neon/Serverless Adapters)
- **Database:** PostgreSQL 17
- **Cache/Sessions:** Redis 7
- **Validation:** Zod (Type-safe schemas)
- **Auth:** JWT (jose) + TOTP 2FA (backup codes)
- **Email:** Postmark/Resend/AWS SES (multi-provider support)
- **Payments:** Stripe (subscriptions) + PIX (PagueBit)
- **Logging:** Winston (structured JSON)
- **Testing:** Vitest + Supertest

## Frontend
- **Admin Panel:** Next.js 16 App Router (port 3000)
- **Tenant App:** Next.js 16 App Router (port 3001)
- **Docs:** Nextra 4 (port 3002)
- **Design System:** @kaven/ui-base (Radix UI + Tailwind CSS 4)
- **State:** Zustand + React Query
- **Styling:** Tailwind CSS 4 (Zero-runtime engine)
- **i18n:** next-intl

## Infrastructure
- **Package Manager:** pnpm 9+
- **Build System:** Turborepo
- **Containerization:** Docker Compose (13+ services)
- **Monitoring:** Prometheus + Grafana + Loki + Promtail

## Security Architecture
- **Middleware layers (10):** CORS → Helmet → CSRF → Rate Limit → Auth → Tenant → RBAC → Capability → IDOR → Business Logic
- **Multi-tenancy:** Row-Level Security via Prisma middleware (`tenantId` obrigatório)
- **Feature flags:** 48 capabilities (35 boolean + 13 numeric) por tenant
- **Audit:** AuditLog com callerType (HUMAN | AGENT | SYSTEM | WEBHOOK)
- **AI Agents:** ServiceToken auth via `X-Service-Token` header (SHA-256 hashed, scoped capabilities)
