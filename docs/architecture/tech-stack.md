# Tech Stack — Kaven Framework

> Generated by `kaven init` — Kaven Framework v1.0.0-rc1

## Backend
- **Runtime:** Node.js 20 LTS
- **Framework:** Fastify 5 (REST API, port 8000)
- **ORM:** Prisma 5
- **Database:** PostgreSQL 17
- **Cache/Sessions:** Redis 7
- **Validation:** Zod + TypeBox (Fastify JSON Schema)
- **Auth:** JWT (accessToken 15m + refreshToken 7d) + TOTP 2FA
- **Email:** AWS SES (rate limited, bulk, templates, SNS webhooks)
- **Payments:** Stripe (subscriptions) + Paddle (marketplace)
- **Logging:** Pino (structured JSON)
- **Testing:** Vitest + Supertest

## Frontend
- **Admin Panel:** Next.js 16 App Router (port 3000) — `apps/admin/`
- **Tenant App:** Next.js 16 App Router (port 3001) — `apps/tenant/`
- **Docs:** Nextra (port 3002) — `apps/docs/`
- **Design System:** @kaven/ui-base (76+ components, Radix UI, shadcn/ui, Frost theme)
- **State:** Zustand (client) + React Query (server state)
- **Styling:** Tailwind CSS 3 + CSS custom properties (Frost tokens)
- **i18n:** next-intl (en-US, pt-BR, es-ES)

## Infrastructure
- **Package Manager:** pnpm 8 + workspaces
- **Build System:** Turborepo
- **Containerization:** Docker Compose (13+ services)
- **CI/CD:** GitHub Actions
- **Database Migrations:** Prisma Migrate
- **Monitoring:** Prometheus + Grafana + Loki + Promtail

## Security Architecture
- **Middleware layers (10):** CORS → Helmet → CSRF → Rate Limit → Auth → Tenant → RBAC → Capability → IDOR → Business Logic
- **Multi-tenancy:** Row-Level Security via Prisma middleware (`tenantId` obrigatório)
- **Feature flags:** 48 capabilities (35 boolean + 13 numeric) por tenant
- **Audit:** AuditLog com callerType (HUMAN | AGENT | SYSTEM | WEBHOOK)
- **AI Agents:** ServiceToken auth via `X-Service-Token` header (SHA-256 hashed, scoped capabilities)
