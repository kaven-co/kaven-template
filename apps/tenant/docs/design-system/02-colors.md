# Cores - Brandbook v2.0.1

## Paleta Primária

### Brand Primary (Verde Esmeralda)
- `#10B981` (main)
- `#059669` (hover/dark)
- `#34D399` (light)

Uso: CTAs principais, highlights, elementos de marca.

### Brand Secondary / Transform (Laranja)
- `#F97316` (main)
- `#EA580C` (hover/dark)
- `#FB923C` (light)

Uso: estados de transformação, destaque contextual e narrativa de customização.

### Brand Accent (Ciano)
- `#00D9FF` (main)

Uso: info, links especiais, transições e pontos de atenção.

## Neutros

- `#0A0A0A`, `#1A1A1A`, `#2A2A2A`, `#4A4A4A`, `#6A6A6A`, `#9A9A9A`, `#CACACA`, `#E5E5E5`, `#F9F9F9`, `#FFFFFF`

## Feedback

- Success: `#10B981`
- Warning: `#F59E0B`
- Error: `#EF4444`
- Info: `#00D9FF`

## Gradientes oficiais

```css
/* Main */
linear-gradient(135deg, #10B981 0%, #00D9FF 50%, #F97316 100%);

/* Transform */
linear-gradient(90deg, #10B981 0%, #F97316 100%);
```

## Do / Don't

### Do
- Usar verde como cor primária da marca.
- Usar laranja/ciano como complementares semânticos.

### Don't
- Não usar azul/roxo como primária padrão.
- Não introduzir hex fora da paleta sem justificativa técnica.
