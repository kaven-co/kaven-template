# Átomos

## Componentes

- Button
- Input
- Label
- Icon
- Badge
- Avatar
- Spinner
- Typography
- Card (base visual)

## Matriz obrigatória

### Button
- 7 sizes × 4 variants (`solid`, `outline`, `ghost`, `text`) × 6 tones

### Input
- 7 sizes × 3 variants (`default`, `filled`, `ghost`) × estados (`default`, `focus`, `error`, `disabled`, `readOnly`)

### Badge
- 6+ sizes × 6 tones × 2 emphasis (`soft`, `solid`)

## A11y mínima

- Botões icon-only com nome acessível
- Focus visível com `focus-visible`
- Labels associadas a inputs
