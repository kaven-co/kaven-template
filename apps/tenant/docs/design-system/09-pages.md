# Páginas

## Papel das páginas

Páginas são composições finais de templates + organismos + dados reais.

## Preview pages

Use páginas de preview para validar regressão visual em:

- tamanhos extremos (`2xs`, `2xl`)
- densidades diferentes
- estados de erro/sucesso/loading
- light/dark

## Requisito

Toda nova feature visual deve ter ao menos um preview de página para validação manual.
