# Tipografia

## Famílias oficiais

- **Display / Headlines / Logo**: Space Grotesk
- **Body / UI / Forms**: Inter
- **Código**: Source Code Pro (ou equivalente monospace)

## Escala recomendada

- Display XL/L/M
- H1/H2 (Space Grotesk)
- H3-H6 (Inter semibold)
- Body L/Body/Body S/Caption (Inter)

## Responsividade

- Reduzir tamanhos display em mobile.
- Manter `16px` como base do body para legibilidade.

## Regra prática

- Evite misturar mais de 3 pesos na mesma tela.
- Títulos sempre em display font.
- Componentes de formulário e tabelas em body font.
