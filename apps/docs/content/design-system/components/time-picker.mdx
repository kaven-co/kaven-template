# Time Picker

> Componente para seleção de horário.

## Básico

```tsx
import { TimePicker } from '@kaven/ui-base';

<TimePicker
  value={time}
  onChange={setTime}
  placeholder="Selecione um horário"
/>;
```

## API

| Prop          | Tipo                   | Padrão  | Descrição            |
| ------------- | ---------------------- | ------- | -------------------- |
| `value`       | `Date \| string`       | -       | Horário selecionado  |
| `onChange`    | `(time: Date) => void` | -       | Callback quando muda |
| `format`      | `'12h' \| '24h'`       | `'24h'` | Formato de hora      |
| `showSeconds` | `boolean`              | `false` | Mostrar segundos     |
