# Button Group

> Agrupa botões relacionados em um único componente.

## Básico

```tsx
import { ButtonGroup } from '@kaven/ui-base';
import { Button } from '@kaven/ui-base';

<ButtonGroup>
  <Button>Opção 1</Button>
  <Button>Opção 2</Button>
  <Button>Opção 3</Button>
</ButtonGroup>;
```

## API

| Prop          | Tipo                                  | Padrão         | Descrição         |
| ------------- | ------------------------------------- | -------------- | ----------------- |
| `variant`     | `'contained' \| 'outlined' \| 'text'` | `'outlined'`   | Estilo dos botões |
| `orientation` | `'horizontal' \| 'vertical'`          | `'horizontal'` | Orientação        |
