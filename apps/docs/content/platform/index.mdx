## 📦 Stack Tecnológica (v1.1.0-alpha)

### Frontend

- **Framework**: Next.js 16 (App Router)
- **UI**: React 19 + Tailwind CSS 4 (Zero-runtime)
- **Estado**: TanStack Query + Zustand
- **i18n**: next-intl

### Backend

- **Runtime**: Node.js 20 LTS
- **Framework**: Fastify 5.8 (Performance-first)
- **ORM**: Prisma 7 (Serverless Adapters)
- **Banco de Dados**: PostgreSQL 17
- **Filas**: BullMQ + Redis 7

### DevOps & Observability

- **Containerização**: Docker Compose (13 services)
- **Monitoramento**: Prometheus + Grafana + Loki
- **Logs**: Winston Structured Logging
- **CI/CD**: GitHub Actions

## 🎯 Features Principais (v2.0)

- ✅ **Kaven CLI** - Gestão completa via linha de comando
- ✅ **Modularidade** - Adicione ou remova features sob demanda
- ✅ **Split-Schema** - Evolução segura do banco de dados
- ✅ **Multi-tenancy** - Isolamento completo de dados
- ✅ **Authentication** - JWT + Refresh Tokens
- ✅ **RBAC** - Controle de acesso baseado em roles
- ✅ **Observability** - Métricas, logs e traces integrados
